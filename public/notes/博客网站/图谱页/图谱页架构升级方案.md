# 图谱页架构升级方案

> 引入 d3-force 物理引擎 + react-force-graph-2d，替换手写 ForceGraph，达到 Quartz V4 级别的 144Hz 丝滑体验。

## 目标

- 解决节点增多后的卡顿问题
- 达到与 Obsidian Graph View / Quartz 一致的缩放流畅度
- 在保持现有 UI 风格的前提下，升级物理模拟内核

## 架构思路：「提取核心引擎，打造定制外壳」

```
┌────────────────────────────────────────────┐
│            顶层渲染框架                      │
│    react-force-graph-2d (React 原生)        │
│    → 保留羊皮纸 UI、React 路由生态            │
├────────────────────────────────────────────┤
│            底层物理内核                      │
│    d3-force (Barnes-Hut 四叉树)             │
│    → O(NlogN) 斥力计算，144Hz 丝滑           │
└────────────────────────────────────────────┘
```

### 底层物理内核（提取自 Quartz）

| 维度 | 旧方案（手写 ForceGraph） | 新方案（d3-force） |
|------|--------------------------|---------------------|
| 斥力算法 | 暴力 O(N²) | Barnes-Hut O(NlogN) |
| 缩放流畅度 | 节点多时掉帧 | 144Hz 无感知卡顿 |
| 物理碰撞 | 简单斥力 | 完整的多体模拟 |
| 参考来源 | 自研 | 与 Obsidian/Quartz 同款引擎 |

### 顶层渲染框架（React 原生定制）

| 维度 | Quartz 方案 | 本方案 |
|------|------------|--------|
| 渲染方式 | Vanilla JS + PixiJS | react-force-graph-2d |
| React 集成 | 需要桥接层 | 原生 React 组件 |
| UI 自定义 | Canvas API 手绘 | nodeCanvasObject 回调 |
| 现有路由 | 无关 | 完美保留 |

### 视觉创新（超越 Quartz）

读取站内 Atlas.jsx 的分类配置，为图谱赋予语义色彩：

| collection | 颜色 | 含义 |
|------------|------|------|
| travel | 绿色 | 旅行路线 |
| project | 金色 | 项目 |
| note | 蓝色 | 笔记 |
| deploy | 棕色 | 建站部署 |
| design | 紫色 | 设计笔记 |

从单调单色网络 → 星系般绚烂的彩色图谱。

| `package.json` | MODIFY | 新增 `react-force-graph-2d` 依赖 |
| `GraphView.jsx` | **重写** | 删除 500+ 行手写 ForceGraph class |
| `MiniGraph.jsx` | MODIFY | 升级为 react-force-graph-2d，紧凑布局 |

## 详细改动

### 1. package.json

```bash
npm install react-force-graph-2d
```

### 2. GraphView.jsx 重构

**删除**：
- 手写的 `ForceGraph` class（约 500 行）
- 暴力的 O(N²) 斥力循环
- 手写的 Canvas 绘制逻辑

**引入**：
- `<ForceGraph2D />` 组件，配置 `d3Force`
- 斥力 `charge` 和连线引力 `link distance` 参数对齐目标效果

**节点色彩化**：
- 根据 `node.collection` 字段分配颜色
- travel → `#4a9`、project → `#da4`、note → `#48e` 等

**Hover 交互**：
- 通过 `nodeCanvasObject` API 重写 Canvas 绘制
- hover 时：当前节点放大、邻居节点高亮、其余节点降低 opacity

### 3. MiniGraph.jsx 升级

- 同样引入 `<ForceGraph2D />`
- 配置更紧凑的引力和更小的连线距离
- 适配小面板尺寸
