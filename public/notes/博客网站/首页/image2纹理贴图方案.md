# 首页 - image2 纹理贴图方案

> 用 AI 生成纹理贴图 + Three.js 动态材质 + 低模几何体，是一条非常有效的视觉升级路线。

## 核心发现

通过 image2 生成海洋纹理，再结合 Three.js shader / 材质系统，效果远好于纯 vertexColors 或程序色块。

## 技术管线

```
image2 生成风格化贴图
    +
Three.js 低模模型承载
    +
shader / 动态材质增强
```

## 各生态区素材需求

### 海洋（✅ 已验证效果好）

| 素材 | 用途 |
|------|------|
| `ocean_albedo` | 基础颜色 |
| `ocean_noise` | 水波扰动 |
| `ocean_foam_mask` | 海岸泡沫 |

目标效果：深海 · 浅海 · 海岸泡沫 · 水波纹理 · 动态流动 · 明亮但不塑料

### 森林 / 雨林（🔲 待生成）

| 素材 | 用途 |
|------|------|
| `forest_ground_albedo` | 地面底色 |
| `leaf_texture` | 叶片纹理 |
| `forest_noise` | 森林噪声 |
| `tree_bark_texture` | 树皮质感 |

目标效果：清新 · 绿意充足 · 叶片层次 · 避免纯绿色方块感
