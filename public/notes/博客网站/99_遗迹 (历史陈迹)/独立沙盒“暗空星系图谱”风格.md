# 实施方案：独立沙盒“暗空星系图谱”风格设计

新建一个独立的测试页面 `GraphConstellationTest.jsx`，在不破坏现有图谱功能的前提下，单独打磨一套星空主题的知识网络星系图。

## 目标与创意设计

### 1. 🌌 星空背景与天体网络
- **沙盒路由**：在 `App.js` 中挂载测试路由 `/graph-test`。
- **深色星河底色**：容器背景重构为深邃星系绿黑底色（`#040806` 到 `#08100d` 的径向渐变），叠加极细的古雅黄金天球仪经纬格线（`rgba(231, 199, 126, 0.04)`）。

### 2. 🌟 节点重构为“发光星云/星核”
- **双层 Canvas 绘制机制**：
  - **内层（星核）**：绘制小半径的高亮白光核心（模拟刺破夜空的真实恒星）。
  - **外层（星晕）**：根据笔记分类（如旅游-金黄、建站-青绿、设计-星宇紫）生成径向发光光晕（`ctx.createRadialGradient`），使节点产生浮动在太空中的恒星质感。
  - **轨道环（Orbital Rings）**：对被 hover 的星体及邻近节点，用 Canvas 绘制**同心虚线轨道环**（`ctx.setLineDash([2, 2])`），营造占星仪仪轨的感觉。

### 3. ✨ 连线重构为“星轨能量链”
- **普通星轨**：极细的星空灰半透明连线（`rgba(231, 199, 126, 0.12)`）。
- **流光能量链**：当 Hover 激活星体时，连接线亮起为耀眼的分类主题色，并增加 `shadowBlur` 营造能量发光管道效果。其他未关联连线和星体淡出（`opacity: 0.15`），使焦点极度突出。

### 4. ✍️ 金沙投影标签 (Celestial Labels)
- 节点文字改用柔和的奶油金沙色（`#ffedd5`），并在绘制文本时加入黑影投影（`ctx.shadowColor = 'rgba(0,0,0,0.85)'`、`ctx.shadowBlur = 4`），确保在任何星系发光背景下都具有极高清晰度。
