# 图谱页星空重构设计方案

> [!NOTE] 航图笔记
> 本册陈列于「建站工坊」的魔药罐（未来构想魔药）中，属于《松果书屋未来功能创意与体验打磨计划》的子专题。
> 主要用于指导后续将图谱页（星纬航图）重塑为“暗空星系”视觉风格。

---

## 🌌 一、 恒星级节点模型 (Star Node Model)

目前的节点只有简单色块，重构方案将其升级为自发光的**恒星天体模型**：
1. **星体分级 (Star Classification)**：
   - 根据手札的被引次数（Backlinks）将节点大小做二次方对数映射，分为巨星、主序星、白矮星，体现手札的权重差异。
2. **气态光晕效果 (Glow & Nebula)**：
   - 每一个节点使用 CSS `filter: drop-shadow` 或 Canvas 的多层 `shadowBlur` 渲染发光层。
   - 处于同一 Collection（分类）的笔记节点会共享一个极淡的径向彩色渐变底图，形成“星座星云”的视觉聚拢感。

---

## 🧭 二、 双链引力轨线 (Wikilink Gravity Lines)

- **流光线效 (Glowing Links)**：
  - 连线不再是单调的实线，而是采用 SVG 虚线。通过给 `stroke-dasharray` 绑定 CSS 动画让微弱的光点在连线上以一定方向缓缓流动，指引链接的引用关系流向。
- **动态引力分布 (Force Directed Control)**：
  - 加载力导向图（D3-Force）时引入缓动摩擦力，使节点在拖拽时有如漂浮在深空中受到万有引力拉扯的缓慢摆动感，而不是干瘪的线性反弹。

---

## 🚀 三、 空间跃迁导航 (Constellation Flight)

- **视角推进动画 (Camera Fly-to)**：
  - 当读者在图谱中双击任意笔记恒星时，屏幕触发“飞船跃迁”效果：整个星空背景在短时间内沿 z 轴方向急速拉伸并虚化，相机视角以三次贝塞尔曲线快速平滑地聚焦到该笔记中心。
  - 聚焦后，该节点的关联节点被依次点亮，不相关的恒星半透明淡出，彻底清除杂乱的节点视线干扰。
- **星空仪控制罗盘 (Translucent Controls)**：
  - 在页面角落设计一个具有玻璃拟态（Glassmorphism）质感的圆形罗盘。
  - 支持调节“引力舒张度”（即控制排斥力参数，使星图膨胀或收缩）、“星等过滤”（只显示核心大词），并可以按标签（如 #嵌入式 #编译原理）高亮特定星座。
