# 建站工坊造物架与小玩具归档计划

> [!NOTE] 造物清单
> 本册陈列于「建站工坊」的魔药罐（未来构想魔药）中，属于《松果书屋未来功能创意与体验打磨计划》的子专题。
> 专门解决工坊目前点击项单一的“简陋”痛点，将屋主在建站与修行过程中积累的各种小项目、小玩具进行集中归档，并以可交互的“蓝图架”形式在工坊展现。

---

## 🛠️ 一、 扩建方案：造物图纸架 (Showcase Shelf)

- **物理交互组件**：
  - 在工坊工作台左侧空白处，增加一个带有铜质发条与皮带滑轮的 **“造物图纸架 (Showcase Shelf)”**。
  - 点击后，图纸架的发条旋转并播放发条拉伸的声效，随之弹出一个具有羊皮纸图纸风格的独立弹窗。
- **图纸卡片设计**：
  - 弹窗中采用网格展示各个小项目卡片。
  - 每一张卡片设计为“工程图纸”质感：半透明黑金配色，四周带有刻度线。包含：
    - **项目名称与代号**（如：代号《织墨》）
    - **当前造物状态**：构思中（草稿）、开发中（炼成中）、已完成（已出厂）、实验中（充能中）
    - **法术核心（技术栈）**：React / Three.js / Node.js
    - **交互法阵 (Action Buttons)**：
      - `[尝试法阵]` (Live Demo iframe 或外部链接跃迁)
      - `[翻阅日志]` (直接在阅读器中打开该项目的 Obsidian 记录文档)

---

## ⚙️ 二、 收录小项目与玩具清单

工坊图纸架将首批集中引入并封装以下 5 个有趣的小玩具与代码实验：

### 1. 3D 行旅轨迹探索器 (3D Route Map)
- **背景说明**：已有的 3D 地图实验（Three.js），在 3D 空间下生成杭州旅行路线的微缩三维轨迹模型。
- **引入逻辑**：提供 Live Demo 入口，点击后图纸弹窗淡出，升起全屏 WebGL 投影进行手势缩放与路线浏览。
- **关联文档**：`博客网站/项目/3D旅游路线地图`

### 2. 现实地图魔改小说世界观 (Fiction World Generator)
- **背景说明**：基于 Mapbox/OpenLayers 地图，在杭州及周边真实地形上叠加小说的修真门派图层。
- **引入逻辑**：内置生成器面板，游客选定西湖、乌龟潭或郭庄，系统会自动吐出该地在小说背景中的“结界等级”、“护山法阵”以及一段 AI 润色的修仙简介小玩具。
- **关联文档**：`博客网站/项目/现实地图魔改小说世界观`

### 3. Obsidian 图谱自动化整理工具 (Obsidian Graph Cleanup Script)
- **背景说明**：用 Node 编写的一个本地图谱净化工具，可扫描库中的死链、修复不规范的 alias 并自动调整引力权重。
- **引入逻辑**：提供 GitHub 源码一键跃迁按钮，并展示该工具运行时的可视化逻辑动图。
- **关联文档**：`博客网站/项目/Obsidian图谱整理`

### 4. WebGL 着色器视觉沙盒 (WebGL Shader Playground)
- **背景说明**：个人练习 GLSL shader 编写的小型视觉合集，包含发光水体、扭曲传送门、动态星空背景。
- **引入逻辑**：在项目卡片中以 300x200 的 Canvas iframe 直接渲染实时着色器，成为工坊最夺目的“法阵演练屏”。
- **关联文档**：`博客网站/项目/Three.js视觉实验`

### 5. 手势控星仪 · 树莓派嵌入式交互实验台 (Gesture Planetarium)
- **背景说明**：课程设计项目——用树莓派摄像头识别手掌骨架，手指张开则三维星球放大、手指合拢则星球缩小。系统分为树莓派识别端（MediaPipe 手势骨架 + Python 参数计算）和网页显示端（Three.js 星空粒子球 + WebGL 渲染），两端通过 WebSocket 传递缩放参数，不传视频流。这方案的精妙之处在于：树莓派那小身板根本跑不动三维渲染，所以只让它做轻量识别和参数输出，重活全扔给浏览器 WebGL——**分工明确，谁也不拖谁后腿**。
- **关联文档**：`系统性知识整理/嵌入式/树莓派手势识别开题报告`、`基于树莓派手势识别的三维星球交互控制系统设计`
- **引入逻辑**：卡片上显示一颗微缩的线框星球。如果树莓派在线且摄像头就绪，滚动条或呼吸灯会显示”神识已连接（Online）”，并展示最近一次手势采样数据（指尖距离、缩放比例）。点击 `[运转法阵]` 可跃迁至网页端实时演示页面——对着摄像头张开手掌，星球就放大；合拢手掌，星球就缩小。纯直觉交互，不需要说明书。
