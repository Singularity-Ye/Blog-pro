## 交互行为

- hover：书本微微抬起，发光
- 点击：书本打开，展示项目详情
- 拖拽：可以拿起书本（物理动画）
- 长按：书本翻页展示项目截图

## 代码映射

交互物件的路径和项目映射已集中在：

→ `地图插件/sc-datav-main/src/constants/blogAssets.js`

```js
import { OBJECT, OBJECT_CONTENT, getAssetPath } from '@/constants/blogAssets';

// 旅行地图册 hover 状态
const path = getAssetPath('mapbook', 'hover'); // → /assets/blog/object/mapbook/hover.png

// 物件对应的项目页面
const { project, route } = OBJECT_CONTENT.mapbook;
```

交互物件图片存放位置：`博客网站/博客页/松果书屋设计素材/交互物件图/`
