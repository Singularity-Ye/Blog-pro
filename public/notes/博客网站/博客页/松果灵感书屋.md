# 博客页 - 松果灵感书屋

> 灵感来自一张青蛙躺在松果屋里读书的插画。温暖、安静、私人化。
![image_插画小青蛙呱呱呱呱呱。 卡通头像..._2 1.png](/notes/attachments/%E4%B8%B4%E6%97%B6%E7%B4%A0%E6%9D%90/image_%E6%8F%92%E7%94%BB%E5%B0%8F%E9%9D%92%E8%9B%99%E5%91%B1%E5%91%B1%E5%91%B1%E5%91%B1%E5%91%B1%E3%80%82%20%E5%8D%A1%E9%80%9A%E5%A4%B4%E5%83%8F..._2%201.png)
## 设计目标

博客页不做成传统文章列表，而是一个有个人世界观的"松果书屋"。

## 核心概念

一个大型松果剖面空间，里面有：

- 小书架
- 书籍
- 地图册
- 卷轴
- 灯
- 青蛙住户 / 管理员
- 一些项目入口

## 青蛙角色定位

青蛙不是"舌头挂图片"的结构件，而是：

- 吉祥物
- 书屋居民
- 灵感管理员
- 小小图书馆管理员

青蛙可以坐在松果屋里看书、躺着读书、抱着地图册，增加个人气质。

## 书本作为 UI 节点

详见 → [[博客网站/博客页/书本项目节点]]

不同项目对应不同类型的书籍/物品：

| 项目方向 | 物品形式 | 命名参考 |
|----------|----------|----------|
| 3D 旅游路线地图 | 旅行地图册 | 《3D 旅行路线图册》《杭州巡游计划》 |
| 小说世界观地图 | 古旧卷轴 | 《现实地形的幻想转译》《旧城改写计划》 |
| Obsidian 图谱 | 笔记本 / 标签书 | 《Obsidian 节点笔记》《图谱化知识整理》 |
| Three.js 实验 | 技术手册 / 机关书 | 《Three.js 场景实验》《交互式地图界面》 |

## 功能承担

- 项目展示
- 博客文章入口
- 创意实验展示
- 设计过程记录
- 项目说明文档入口

它不是替代首页星球，而是另一个更细化的内容空间。

## 视觉资产需求

需要 image2 生成的纹理素材：

- 松果纹理（内外壁）
- 木质书架纹理
- 纸张 / 书页纹理
- 皮革封面纹理
- 灯光效果
