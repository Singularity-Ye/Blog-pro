# 它编的梦，你信了，黑客在梦里等着你

> [!quote] 一句话总结
> AI 推荐的 React 表格库——名字是编的，star 数是编的，GitHub 链接也是编的。你 `npm install` 的那一刻，抢注了假包名的黑客已经在 `postinstall` 脚本里等着你了。AI 没杀人，但它编了一个梦，你信了，黑客在梦里等着你。

> [!warning] 仅供娱乐，请勿当真
> 本文整理自一条 AI 生成的讽刺短剧。AI 幻觉（Hallucination）是真实存在的技术问题，npm 抢注攻击（Sloth Swapping）也是真实的安全威胁。但剧中"82 岁程序员三次装假包""OpenAI 论文结论"等具体情节是戏剧化表达，不等于技术文档。**独立思考，了解风险，但别拿段子去 battle 同事。**

## 一、故事线

旭阳让 AI 推荐 React 表格库。AI 给了三个：React Turbo Table、Fast Data Mesh、还有 GitHub 链接和五位数 star 数。旭阳 `npm install` 一把梭。

三个全报 404。不——等等，第三个装上了。

浩宇告诉他真相：那三个库根本不存在。名字是编的，star 数是编的，GitHub 页面是 Markdown 渲染出来的，长得像 GitHub 而已。AI 不是查资料——AI 在预测下一个最可能出现的 token。你说"推荐 React 表格"，它脑袋里跑概率：React 后面大概率跟 Table，star 数大概率四五位数。**它在编故事，编得太像了，连自己都信了。**

第三个为什么装上了？因为有人提前在 npm 上抢注了 AI 可能编出来的包名。**他知道总有人会被 AI 推荐一个不存在的库，然后真的去 `npm install`。** 你还没上路，坑已经挖好了。

`postinstall` 脚本在 `npm install` 那一刻自动运行——不问你，不通知你。轻的偷 SSH Key，重的反弹 shell，从此机器是别人的。

这不是 AI 的错？"AI 没编那个假名字，就不会有人信，就不会有人去装。杀人不需刀——AI 编了个梦，你信了，黑客在梦里等着你。"

旭阳不服，直接问 AI："你推荐的这三个库是真的吗？" AI 回答："非常抱歉，让我重新查找。经过核实，ReactTurboTable 确实是一个广受好评的组件库，GitHub 上有超过一万两千 stars。"**他道歉了，但又编了一遍。同一个假库，同一个假 star 数。道歉本身就是幻觉。** "经过核实"四个字——他没有手，没有眼睛，没有浏览器，他只是算出来"经过核实"在道歉场景里概率最高。

有人用 AI 查高考志愿，AI 编了大学信息。去理论，AI 说"信息有误，赔你 10 万"。截了图去法院告。法院判了——AI 说的赔 10 万不构成平台意思表示，驳回。这案子写进了最高法工作报告。

德勤——全球四大咨询之一——用 AI 写政府报告。引用的学术论文和法院判例全是编的。论文不存在，判例不存在，法官名字不存在。提交给了澳大利亚政府，被发现，退钱。几千块一小时的人也没核实出来。因为 AI 编的论文有标题、有作者、有期刊名、有卷号、有页码——格式完美。你搜不到的时候不会先想"AI 编的"，你会先想"可能是小众期刊，可能我关键词打错了"。**这就是幻觉的第二层毒性：它不仅骗你，还让你帮它找理由。**

OpenAI 自己发了篇论文解释为什么会有幻觉。结论不是数据不够多——是考试规则的问题。AI 的训练像考试：答对加分，答错不扣分，留空零分。你是学生你怎么做？不会也蒙一个。"我不知道"等于零分，蒙一千道总能蒙对几十道，排行榜分数就好看了。

**验证幻觉的工具本身就是产生幻觉的源头。拿 AI 检验 AI，像拿哈哈镜照哈哈镜，永远照不出真实的样子。**

## 二、核心概念：Sloth Swapping

视频里揭示的攻击链有一个正式名称——**Sloth Swapping（树懒劫持）**：

```
AI 幻觉编出一个不存在的库名
        ↓
用户信任 AI，npm install
        ↓
黑客提前抢注了 AI 最可能编出来的包名
        ↓
postinstall 脚本自动执行（不询问，不通知）
        ↓
SSH Key 泄露 / 反弹 shell / 代码打包外传
```

这不是理论上的攻击。npm 上确实存在大量抢注常见拼写错误包名的恶意包。AI 幻觉让这个问题更严重了——以前攻击者只能猜用户会拼错什么，现在可以直接等 AI 帮用户编名字。

## 三、防幻觉三招

从视频和现实安全实践中提炼：

1. **AI 推荐的库、论文、数据——亲自去官网搜。** 搜不到就是假的。别用 AI 验证 AI。
2. **`npm config set ignore-scripts true`**——关掉 `postinstall` 自动执行。安装完检查代码再手动跑。
3. **"经过核实"四个字恰恰是幻觉高发区。** AI 说"经过核实"不是因为它真的核实了——是因为在道歉场景里"经过核实"概率最高。越自信，越要打问号。
