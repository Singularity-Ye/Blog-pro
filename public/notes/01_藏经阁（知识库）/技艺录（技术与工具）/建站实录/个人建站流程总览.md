# 个人建站流程总览

> 2026-05-21 实操版。这次不是教程翻译，是亲手部署后积累的真实经验。

## 本次部署结果

```
GitHub (Singularity-Ye/Blog-pro)
  → Cloudflare Pages 构建（CRA → build/）
    → 阿里云 DNS 加 CNAME 记录
      → Cloudflare Pages 绑定自定义域名
        → HTTPS 访问成功
```

| 地址 | 状态 |
|------|:--:|

## 本次 DNS 方案

**阿里云继续管理 DNS**，只加一条 CNAME 指向 Pages 默认域名。

不是"把 DNS 整体交给 Cloudflare"的路线。详见 → [[建站流程指南-静态网页/域名与DNS/阿里云域名 CNAME 绑定 Cloudflare Pages]]

## 本次核心经验

1. **CRA → build，Vite → dist**，不要搞混输出目录
2. **Active + SSL ≠ 网站能打开**，先查 Pages 默认域名
3. **修改配置后要 Retry deployment**，旧部署不会自动更新
4. **阿里云 CNAME + Cloudflare Pages** 是一种合法的轻量方案
