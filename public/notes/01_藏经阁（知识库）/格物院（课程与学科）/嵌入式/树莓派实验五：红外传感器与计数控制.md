# 树莓派实验五：红外传感器与计数控制

> 用红外避障传感器检测物体靠近，控制 LED 亮灭，并实现挥手计数与两次一周期输出。

---

## 一、实验要求

1. 按键（红外、光敏、声控）控制小灯亮灭
2. 按键计数输出，计数结果不同得到不同输出
3. 以上均需有图有程序

最终重点完成：**红外传感器测试**、**红外控制 LED**、**红外挥手计数**、**两次一周期输出**。

---

## 二、红外模块识别

使用的蓝色模块是**红外避障传感器**。

模块前面有：
- **透明管**：红外发射管
- **黑色管**：红外接收管

### 工作原理

```
红外发射管 → 发出红外光 → 前方有物体时反射 → 接收管收到 → 比较器输出电平变化
```

### 实测结论

| 状态 | 输出 |
|------|------|
| 手靠近 / 握住探头 | 0 |
| 远离 / 无物体 | 1 |

> **低电平有效**：检测到物体 = 0，没有物体 = 1。

## 四、红外读数测试

### 代码

```python
import RPi.GPIO as GPIO
import time

IR = 24

GPIO.setwarnings(False)
GPIO.setmode(GPIO.BCM)
GPIO.setup(IR, GPIO.IN)

try:
    while True:
        value = GPIO.input(IR)
        print("IR value:", value)
        time.sleep(0.2)

except KeyboardInterrupt:
    pass

finally:
    GPIO.cleanup()
```

### 实验结论

- 远离时输出 **1**
- 靠近时输出 **0**

## 五、红外控制小灯亮灭

### 代码

```python
import RPi.GPIO as GPIO
import time

IR = 24
LED = 5

GPIO.setwarnings(False)
GPIO.setmode(GPIO.BCM)

GPIO.setup(IR, GPIO.IN)
GPIO.setup(LED, GPIO.OUT)

try:
    while True:
        value = GPIO.input(IR)

        if value == 0:
            GPIO.output(LED, GPIO.HIGH)
        else:
            GPIO.output(LED, GPIO.LOW)

        time.sleep(0.1)

except KeyboardInterrupt:
    pass

finally:
    GPIO.cleanup()
```

### 要点

红外模块检测到物体时输出 0，所以：

```python
if value == 0:    # 检测到靠近 → 亮灯
```

## 六、挥手计数问题

### 核心难点

红外模块输出不是单个数字，而是一串连续值：

```
远离：1 1 1 1 1 1
靠近：0 0 0 0 0 0
远离：1 1 1 1 1 1
```

如果看到一次 0 就计数，一次挥手会被算很多次。

### 解决思路

定义一次**完整挥手**为一个状态变迁周期：

```
远离(1) → 靠近(0) → 远离(1)
```

只有完成"从 1 回到 1"这一轮，计数才加 1。

### 去抖动机制

用连续采样去抖：同一个值连续读到 `STABLE_NEED`（8）次，才算一次稳定状态变化。

```python
if same_count >= STABLE_NEED:
    stable_state = value
```

## 七、两次一周期控制 LED

### 逻辑说明

不是"挥手 1 次切灯"，而是**两次挥手为一个单位周期**：

| 挥手次数 | group = count // 2 | LED 状态 |
|---------|-------------------|---------|
| 第 1 次 | 0 | 不亮 |
| 第 2 次 | 1 | 亮 |
| 第 3 次 | 1 | 仍亮 |
| 第 4 次 | 2 | 灭 |
| 第 5 次 | 2 | 仍灭 |
| 第 6 次 | 3 | 亮 |

### 核心代码

```python
group = count // 2

if group % 2 == 1:
    GPIO.output(LED, GPIO.HIGH)   # 亮
else:
    GPIO.output(LED, GPIO.LOW)    # 灭
```

## 八、完整代码

```python
import RPi.GPIO as GPIO
import time

IR = 24
LED = 5

GPIO.setwarnings(False)
GPIO.setmode(GPIO.BCM)

GPIO.setup(IR, GPIO.IN, pull_up_down=GPIO.PUD_UP)
GPIO.setup(LED, GPIO.OUT)

GPIO.output(LED, GPIO.LOW)

count = 0

stable_state = 1
last_stable_state = 1

same_count = 0
last_read = GPIO.input(IR)

STABLE_NEED = 8

print("程序开始：无物体时应为1，靠近时应为0")
print("两次挥手为一个单位周期")
print("第1次不变，第2次亮；第3次不变，第4次灭")

try:
    while True:
        value = GPIO.input(IR)

        if value == last_read:
            same_count += 1
        else:
            same_count = 0
            last_read = value

        if same_count >= STABLE_NEED:
            stable_state = value

        if stable_state != last_stable_state:

            if last_stable_state == 1 and stable_state == 0:
                print("检测到靠近")

            elif last_stable_state == 0 and stable_state == 1:
                count += 1
                print("挥手次数 =", count)

                group = count // 2

                if group % 2 == 1:
                    GPIO.output(LED, GPIO.HIGH)
                    print("当前输出 = 1，小灯亮")
                else:
                    GPIO.output(LED, GPIO.LOW)
                    print("当前输出 = 0，小灯灭")

            last_stable_state = stable_state
            same_count = 0

        time.sleep(0.02)

except KeyboardInterrupt:
    pass

finally:
    GPIO.cleanup()
```

## 九、关键概念：状态机思维

红外计数的核心不是"读到 0 就+1"，而是**跟踪状态变迁**：

```
状态1(远离) ──靠近──→ 状态0(靠近) ──远离──→ 计数+1，回到状态1
```

配合去抖动（连续 8 次相同读数才算稳定），避免一次挥手被重复计数。
