# 树莓派实验七：红外门禁系统

> 红外传感器检测人靠近与远离，连续旋转舵机模拟抬杆放杆，OLED 实时显示累计人数与状态。输入 → 执行 → 显示，完整小系统。

---

## 一、实验目标

用树莓派实现一个简易红外门禁系统：

- 红外传感器检测是否有人靠近
- 检测到靠近时，连续旋转舵机正转，模拟**抬杆**
- 检测到远离时，舵机反转，模拟**放杆**
- 每完成一次"靠近 → 远离"，人数计数 +1
- OLED 实时显示当前人数、状态和时间

整体流程：

```text
无人状态（OUT = 1）
  ↓
红外检测到靠近（OUT = 0）
  ↓
舵机正转，抬杆
  ↓
红外检测到远离（OUT = 1）
  ↓
舵机反转，放杆
  ↓
count + 1，OLED 更新
```

---

## 二、使用模块

| 模块 | 作用 |
|------|------|
| 树莓派 | 主控板，运行 Python |
| 红外避障传感器 | 检测是否有人靠近 |
| 连续旋转舵机 | 模拟门禁栏杆抬放 |
| OLED 显示屏（SSD1306） | 显示人数、状态、时间 |

## 三、引脚连接

> 代码使用 `GPIO.setmode(GPIO.BCM)`，以下 BCM 编号对应树莓派物理引脚。

### 红外传感器

| 红外模块 | 树莓派物理脚 | 功能 |
|---------|-------------|------|
| VCC | 1 | 3.3V |
| GND | 6 | GND |
| OUT | 18 | GPIO24 |

- 靠近/遮挡 = **0**（低电平有效）
- 远离/无人 = **1**

### 连续旋转舵机

| 舵机线 | 树莓派物理脚 | 功能 |
|--------|-------------|------|
| 红 VCC | 2 或 4 | 5V |
| 棕 GND | 9 | GND |
| 橙 Signal | 12 | GPIO18 |

> 舵机用 **5V** 供电，信号线接 GPIO18。舵机 GND 必须与树莓派 GND **共地**，否则信号无法识别。

### OLED 显示屏

| OLED | 树莓派物理脚 | 功能 |
|------|-------------|------|
| GND | 25 | GND |
| VDD | 17 | 3.3V |
| SCK | 5 | GPIO3 / SCL |
| SDA | 3 | GPIO2 / SDA |

```bash
i2cdetect -y 1
# 输出中出现 3c → OLED 地址 0x3C
```

## 四、实验过程与调试记录

### 1. 红外传感器测试

之前已确认输出逻辑：

| 状态 | OUT |
|------|-----|
| 手靠近 / 遮挡 | 0 |
| 远离 / 无遮挡 | 1 |

红外输出是连续变化的（`111111` → `000000` → `111111`），不能"读到 0 就计数"。程序采用**状态变化判断**：

- `1 → 0`：检测到靠近
- `0 → 1`：检测到远离

完整经历一次 `1 → 0 → 1` 才算一次通行。

### 2. 舵机类型判断

最初用普通舵机角度控制方式（`set_angle(90)`），但现象不稳定：

- 转动角度过大
- 正反转不一致
- 只正转不反转

判断当前舵机是**连续旋转舵机**，不能设置角度，而是：

```text
转动角度 ≈ 转速 × 转动时间
```

- **Duty** 控制方向和速度
- **Time** 控制转多久

### 3. 舵机强制测试

用极端占空比确认舵机硬件正常：

```python
Duty = 10.0   # 正转
Duty = 7.5    # 停止
Duty = 5.0    # 反转
```

结论：之前 `8.0 / 8.0` 不转，是因为距离停止点 `7.5` 太近，推力不足。

### 4. 正反转时间校准

| 动作 | 占空比 | 0.30 秒转动角度 |
|------|--------|----------------|
| 正转 | 10.0 | ≈ 180° |
| 反转 | 5.0 | ≈ 120° |

目标 90°，按比例：

- `FORWARD_TIME = 0.15`（正转）
- `BACKWARD_TIME = 0.23`（反转）

---

## 五、核心参数说明

```python
STOP_DUTY     = 7.5   # 舵机停止
FORWARD_DUTY  = 10.0  # 正转（抬杆）
BACKWARD_DUTY = 5.0   # 反转（放杆）
FORWARD_TIME  = 0.15  # 正转时长
BACKWARD_TIME = 0.23  # 反转时长
```

### 调参规律

| 现象 | 调整 |
|------|------|
| 正转角度太大 | 减小 `FORWARD_TIME` |
| 正转角度太小 | 增大 `FORWARD_TIME` |
| 舵机完全不动 | 增大 `FORWARD_DUTY` 或减小 `BACKWARD_DUTY` |
| 停止后仍慢慢转 | 微调 `STOP_DUTY`，如 7.4 / 7.6 |

## 六、最终程序代码

```python
# -*- coding: utf-8 -*-

import os
import fcntl
import time
from datetime import datetime

import RPi.GPIO as GPIO
from PIL import Image, ImageDraw, ImageFont


# =========================
# 一、引脚设置
# =========================

# 红外传感器 OUT 接 GPIO24，物理18号脚
IR = 24

# 连续旋转舵机信号线接 GPIO18，物理12号脚
SERVO = 18


# =========================
# 二、OLED 参数
# =========================

OLED_ADDR = 0x3C
I2C_BUS = 1
WIDTH = 128
HEIGHT = 64
I2C_SLAVE = 0x0703
FONT_PATH = "/usr/share/fonts/truetype/wqy/wqy-zenhei.ttc"


# =========================
# 三、连续旋转舵机参数
# =========================

STOP_DUTY      = 7.5
FORWARD_DUTY   = 10.0
BACKWARD_DUTY  = 5.0
FORWARD_TIME   = 0.15
BACKWARD_TIME  = 0.23

# 红外检测防抖参数
STABLE_NEED = 2
DELAY = 0.01


# =========================
# 四、OLED 原生 I2C 驱动
# =========================

class SSD1306:
    def __init__(self, bus=1, address=0x3C):
        self.address = address
        self.fd = os.open("/dev/i2c-{}".format(bus), os.O_RDWR)
        fcntl.ioctl(self.fd, I2C_SLAVE, address)
        self.init_display()

    def command(self, cmd):
        os.write(self.fd, bytes([0x00, cmd]))

    def data(self, data_bytes):
        for i in range(0, len(data_bytes), 16):
            chunk = data_bytes[i:i + 16]
            os.write(self.fd, bytes([0x40]) + bytes(chunk))

    def init_display(self):
        cmds = [
            0xAE,        # 关闭显示
            0xD5, 0x80,  # 设置显示时钟
            0xA8, 0x3F,  # 复用率 64 行
            0xD3, 0x00,  # 显示偏移
            0x40,        # 起始行
            0x8D, 0x14,  # 电荷泵
            0x20, 0x00,  # 水平寻址
            0xA1,        # 列地址映射
            0xC8,        # COM 扫描方向
            0xDA, 0x12,  # COM 引脚配置
            0x81, 0xCF,  # 对比度
            0xD9, 0xF1,  # 预充电周期
            0xDB, 0x40,  # VCOMH
            0xA4,        # 显存内容显示
            0xA6,        # 正常显示
            0xAF         # 开启显示
        ]
        for cmd in cmds:
            self.command(cmd)
        self.clear()

    def clear(self):
        self.show(Image.new("1", (WIDTH, HEIGHT)))

    def show(self, image):
        image = image.convert("1")
        self.command(0x21)
        self.command(0)
        self.command(WIDTH - 1)
        self.command(0x22)
        self.command(0)
        self.command(7)

        pixels = image.load()
        buffer = []
        for page in range(8):
            for x in range(WIDTH):
                byte = 0
                for bit in range(8):
                    y = page * 8 + bit
                    if pixels[x, y] != 0:
                        byte |= (1 << bit)
                buffer.append(byte)
        self.data(buffer)


# =========================
# 五、GPIO 与 OLED 初始化
# =========================

GPIO.setwarnings(False)
GPIO.setmode(GPIO.BCM)

GPIO.setup(IR, GPIO.IN, pull_up_down=GPIO.PUD_UP)

GPIO.setup(SERVO, GPIO.OUT)
servo = GPIO.PWM(SERVO, 50)
servo.start(STOP_DUTY)

oled = SSD1306(I2C_BUS, OLED_ADDR)

try:
    font_title = ImageFont.truetype(FONT_PATH, 14)
    font_big   = ImageFont.truetype(FONT_PATH, 20)
    font_mid   = ImageFont.truetype(FONT_PATH, 12)
    font_small = ImageFont.truetype(FONT_PATH, 10)
except:
    font_title = ImageFont.load_default()
    font_big   = ImageFont.load_default()
    font_mid   = ImageFont.load_default()
    font_small = ImageFont.load_default()


# =========================
# 六、功能函数
# =========================

def servo_stop():
    """停止舵机"""
    servo.ChangeDutyCycle(STOP_DUTY)
    time.sleep(0.1)


def servo_forward():
    """正转：抬杆"""
    print("靠近：正转 Duty =", FORWARD_DUTY, "Time =", FORWARD_TIME)
    servo.ChangeDutyCycle(FORWARD_DUTY)
    time.sleep(FORWARD_TIME)
    servo_stop()


def servo_backward():
    """反转：放杆"""
    print("远离：反转 Duty =", BACKWARD_DUTY, "Time =", BACKWARD_TIME)
    servo.ChangeDutyCycle(BACKWARD_DUTY)
    time.sleep(BACKWARD_TIME)
    servo_stop()


def read_ir_stable():
    """稳定读取红外状态，防抖"""
    last_value = GPIO.input(IR)
    same_count = 0
    while True:
        value = GPIO.input(IR)
        if value == last_value:
            same_count += 1
        else:
            last_value = value
            same_count = 0
        if same_count >= STABLE_NEED:
            return value
        time.sleep(DELAY)


def draw_person_icon(draw, x0, y0):
    """手搓小人图标"""
    person = [
        "00111100",
        "01111110",
        "01111110",
        "00111100",
        "00011000",
        "01111110",
        "10011001",
        "00011000",
        "00100100",
        "01000010",
    ]
    for y, row in enumerate(person):
        for x, pixel in enumerate(row):
            if pixel == "1":
                draw.point((x0 + x, y0 + y), fill=255)


def update_oled(count, status):
    """更新 OLED 显示"""
    image = Image.new("1", (WIDTH, HEIGHT))
    draw = ImageDraw.Draw(image)
    now = datetime.now().strftime("%H:%M:%S")

    draw.text((0, 0), "红外门禁系统", font=font_title, fill=255)
    draw.text((0, 20), "人数：{}".format(count), font=font_big, fill=255)
    draw.text((0, 46), "状态：" + status, font=font_mid, fill=255)
    draw.text((76, 54), now, font=font_small, fill=255)

    draw_person_icon(draw, 112, 20)

    oled.show(image)


# =========================
# 七、主程序
# =========================

count = 0

try:
    print("红外门禁系统启动")
    print("靠近=0，远离=1")
    print("靠近时抬杆，远离时放杆并计数")
    print("当前参数：")
    print("STOP_DUTY =", STOP_DUTY)
    print("FORWARD_DUTY =", FORWARD_DUTY)
    print("BACKWARD_DUTY =", BACKWARD_DUTY)
    print("FORWARD_TIME =", FORWARD_TIME)
    print("BACKWARD_TIME =", BACKWARD_TIME)
    print("按 Ctrl+C 退出")

    servo_stop()
    update_oled(count, "等待")

    # 读取初始状态，避免启动时误触发
    last_state = read_ir_stable()
    print("初始红外状态 =", last_state)

    while True:
        value = read_ir_stable()

        # 远离 1 → 靠近 0：抬杆
        if last_state == 1 and value == 0:
            print("检测到靠近，抬杆")
            update_oled(count, "开门")
            servo_forward()
            last_state = 0

        # 靠近 0 → 远离 1：放杆并计数
        elif last_state == 0 and value == 1:
            print("检测到远离，放杆")
            servo_backward()

            count += 1
            print("累计人数 =", count)

            update_oled(count, "已通过")
            time.sleep(0.3)
            update_oled(count, "等待")

            last_state = 1

        time.sleep(0.02)

except KeyboardInterrupt:
    print("\n用户中断程序")

finally:
    servo_stop()
    oled.clear()
    servo.stop()
    GPIO.cleanup()
    print("程序结束，GPIO资源已清理")
```

## 七、实验现象

程序运行后，OLED 显示：

```text
红外门禁系统
人数：0
状态：等待
当前时间
```

- **手靠近**：红外 OUT → 0，舵机正转抬杆，OLED 显示"开门"
- **手离开**：红外 OUT → 1，舵机反转放杆，人数 +1，OLED 更新

完整链路：**红外检测 → 舵机动作 → 人数计数 → OLED 显示**

## 八、踩坑与解决

### 舵机不能按角度控制

使用 `set_angle(90)` 方式效果不稳定，判断为连续旋转舵机。

→ 改用占空比 + 时间控制。

### 占空比 8.0 无法驱动

`FORWARD_DUTY = 8.0` 距离停止点 7.5 太近，推力不足。

→ 改为 `10.0 / 5.0`，拉开距离。

### 正反转角度不一致

正转 0.30s ≈ 180°，反转 0.30s ≈ 120°，速度不同。

→ 分别设置时间：正转 0.15s，反转 0.23s。

### OLED 库兼容问题

Adafruit 库在当前 Python 版本中不兼容。

→ 改用原生 I2C 写 SSD1306（沿用实验六方案）。

## 九、实验总结

本实验把红外输入、舵机执行、OLED 显示串成了一条完整链路，从单模块控制升级为**输入 → 执行 → 显示**的小系统。

核心收获：连续旋转舵机不是普通 180° 舵机，不能直接设置角度，而是通过 PWM 占空比控制方向和速度、通过持续时间控制转动幅度的小电机。
