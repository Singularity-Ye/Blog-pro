# TEC-2 原始微程序清单与 MAPROM 入口表

> TEC-2 控制存储器（CM）完整微程序清单、MAPROM 入口地址映射、FS1-3 功能选择器说明。
> 来源：胡敏、钱兴贤《计算机组成接口与通信实验指导》。
>
> 后续设计任何自定义微程序，都应先查此表找相近微操作，再改字段。不要靠肉眼猜字段递增规律。

## 一、FS1-3 功能选择器

| FS1-3 | 功能 |
|:-----:|------|
| 000 | 装入程序 |
| 001 | 执行程序 |
| 010 | 写存储器 |
| 011 | 读存储器（单步） |
| 100 | 读存储器（连续） |
| 101 | 从 0 连续执行 |
| 110 | 单步执行 |
| 111 | 连续执行 |

## 四、使用说明

1. **查相近指令：** 设计新微程序时，先在第三节找功能最接近的已有指令，从它的微码出发修改。
2. **MAPROM 入口：** 未用操作码（C0–FC）的入口地址 110H–1A0H 可用于自定义微程序。
3. **取指/执行对：** 偶数地址为取指周期（通常是 `00 0E00 A0B5 5402`），奇数地址为执行周期。
4. **字段解读需谨慎：** 56 位微指令的字段划分取决于具体硬件设计，不要仅凭十六进制表面变化推断寄存器选择。详见 [[TEC-2 矩阵指令微操作样例表]] 中的踩坑记录。
