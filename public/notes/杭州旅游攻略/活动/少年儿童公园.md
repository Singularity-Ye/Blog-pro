# 杭州少年儿童公园（满陇桂雨公园）

位于虎跑路的儿童主题公园，五一期间有亲子互动活动，园内有小型游乐设施。

## 基本信息

- **位置**：西湖区虎跑路（动物园旁）
- **时间**：全年开放，8:30-17:00
- **门票**：公园免费，部分游乐设施 ¥10-30/次
- **交通**：公交 4 路 / 315 路至动物园站

## 核心内容

- 🎠 小型游乐设施（旋转木马、碰碰车等）
- 🌳 桂花主题园林（秋天最美，但五一绿意盎然）
- 🦜 旁边就是 [[杭州动物园]]
- 🧒 亲子互动区

## 游玩建议

- ⏰ 建议游玩 1.5-2 小时
- 🔗 强烈推荐与 [[杭州动物园]] 串联（步行 3 分钟）
- 🧺 适合带野餐垫在草坪休息
- 🚰 园内有饮水点

## 串联建议

```
少年儿童公园 → [[杭州动物园]] → 虎跑路沿线 → 满觉陇
     (1.5h)         (2h)          (1h)       (0.5h)
```
