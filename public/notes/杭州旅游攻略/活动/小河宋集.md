# 小河宋集

[[小河直街]] 五一期间的宋式美学沉浸体验市集。运河边的文艺街区变身为宋代市井——点茶、焚香、插花、挂画，宋人四雅一次体验。

## 基本信息

- 📅 **5月1日-5日**（全程覆盖五一）
- 📍 [[小河直街]]
- 💰 免费
- 🚇 地铁 5 号线「大运河」站 / 水上巴士至拱宸桥步行 15min

## 体验内容

- 🍵 宋代点茶表演和体验
- 🎨 团扇绘制、香囊手作
- 🎵 古筝/琵琶街头演奏
- 🛍️ 独立设计师文创摊位
- 🍡 宋式茶点小吃

## 🟢 完美嵌入 Day 3 运河线

```
武林门码头 → 水上巴士 → 拱宸桥 → 桥西（运河有戏 5/1-2）
                                    ↓
                              运河博物馆群
                                    ↓
                              小河直街（小河宋集 5/1-5）
                                    ↓
                              🚗 武林夜市
```

## 拍照建议

- 傍晚灯光效果最佳（灯笼+运河倒影）
- 穿汉服/素色衣服更出片
- 河边石板路+老房子背景
