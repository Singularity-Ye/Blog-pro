# 胜利河龙虾啤酒节

胜利河美食街的五一特别活动。市集 + 现场乐队 + 龙虾畅吃 + 啤酒畅饮。杭州本地人夏天最爱的夜宵据点之一。

## 基本信息

- 📅 即日起 — 5月5日
- 📍 拱墅区胜利河美食街
- 💰 免费入场（餐饮另付，人均 ¥80-120）
- 🕐 17:00 - 凌晨 2:00（夜市性质，越晚越热闹）
- 🚇 地铁 3 号线「香积寺」站步行 10 分钟

## 推荐

- 🦞 麻辣/十三香小龙虾
- 🍺 本地精酿啤酒
- 🎵 现场乐队（每晚 19:00 开始）

## 🟢 与运河线完美串联

```
香积寺（¥20，吃素面）→[步行 10min]→ 胜利河龙虾啤酒节（晚餐+夜宵）
```

> 💡 白天逛运河、晚上来吃龙虾——这是杭州本地人的五一正确打开方式。
