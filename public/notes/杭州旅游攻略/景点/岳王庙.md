# 岳王庙

纪念抗金名将岳飞的祠庙。位于 [[苏堤]] 北口对面，栖霞岭南麓。墓前跪着秦桧等四奸臣铁像——"青山有幸埋忠骨，白铁无辜铸佞臣"。这是来杭州必去的人文景点，历史分量极重。

## 基本信息

- **门票**：¥25
- **位置**：北山街 80 号（[[苏堤]] 北口对面）
- **开放时间**：7:00 - 17:30
- **游览时间**：30-60min
- **交通**：公交 7 路/27 路至岳坟站

## 核心看点

### ⚔️ 岳飞故事

| 看点 | 说明 |
|------|------|
| 正殿 | 岳飞坐像，上方"还我河山"匾额 |
| 岳飞墓 | 墓前四跪像（秦桧、王氏、万俟卨、张俊） |
| 精忠柏 | 传说被岳飞精神感化的化石 |
| 碑廊 | 历代名人题咏 |
| 南枝巢 | 岳飞纪念馆，展示岳飞生平 |

> 💡 **"莫须有"三字**——秦桧以"莫须有"罪名害死岳飞，是中国历史上最著名的冤案。来岳王庙不仅是看景点，更是读历史。

### 📸 四跪像

| 人物 | 身份 | 说明 |
|------|------|------|
| 秦桧 | 宰相 | 主谋，跪了 500 年 |
| 王氏 | 秦桧妻子 | "捉虎易，放虎难" |
| 万俟卨 | 御史中丞 | 直接审讯岳飞 |
