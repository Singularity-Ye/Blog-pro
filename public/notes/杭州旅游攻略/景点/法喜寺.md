# 法喜寺（上天竺）

杭州求姻缘最灵的寺庙。网红机位"飞檐长廊"刷爆小红书，5 元素斋更是性价比之王。在杭州年轻人中人气极高——来法喜寺求姻缘、拍照、吃素斋，是杭州"新民俗"。

## 基本信息

- **门票**：¥10（赠送三支清香）
- **开放时间**：6:30 - 18:00
- **交通**：公交 103/1314/121 路「上天竺」站。无地铁直达，最近地铁 3 号线东岳站（约 3.9km）
- **从灵隐寺来**：观光车约 ¥10/人，或步行天竺路约 20min
- **游览时间**：1-2h

## 🟢 串联建议

```
方案 1（灵隐→法喜→龙井）：
灵隐寺（2h）→ 步行/观光车 20min → 法喜寺（1h，吃¥5素斋）
    → 打车 10min → 龙井村 → 九溪十八涧

方案 2（天竺三寺）：
法净寺（免费）→ 步行 5min → 法镜寺（¥10）→ 步行 5min → 法喜寺（¥10）
    （天竺路沿线，全程 1.5h）

方案 3（轻松版）：
法喜寺（9:00，拍照+求签）→ 素斋（11:00 ¥5 午餐）
    → 天竺路散步 → 灵隐寺（下午）
```

## ⚠️ 五一提醒

- 法喜寺五一人比灵隐寺少——但飞檐长廊拍照要排队
- 🟢 建议 **7:00 开门就去**（人最少+光线最好）
- 素斋 ¥5 五一人多可能 11:00 就卖完——早点去
- 文创御守五一可能断货——想要的早买
- 无地铁直达 → 公交 103/1314 路「上天竺」站最方便
- 💡 最佳策略：法喜寺（7:00）→ 素斋（11:00）→ 灵隐寺（下午）
