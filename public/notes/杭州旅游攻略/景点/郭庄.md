# 郭庄

杭州唯一原汁原味的江南古典园林，位于杨公堤。临湖茶室可以**坐着喝茶看西湖**——这在断桥和苏堤上做不到。是西湖秘境线的起点，也是五一避开人潮的最佳选择。

## 基本信息

- **门票**：¥10（需预约，"西湖旅游"小程序）
- **位置**：西湖区杨公堤 28 号
- **开放时间**：7:30 - 17:00
- **游览时间**：30-60min（喝茶的话 1-1.5h）

## 核心看点

### 🏡 园林布局

郭庄分"静必居"和"一镜天开"两部分：

| 区域   | 特点      | 看点        |     |
| ---- | ------- | --------- | --- |
| 静必居  | 住宅区，私密感 | 曲廊、天井、假山  |     |
| 一镜天开 | 临湖区，开阔  | 水榭、湖景、荷花池 |     |

### 🍵 临湖茶室

- **位置**：一镜天开水榭旁
- **茶价**：¥40-80/杯（龙井、碧螺春等）
- **体验**：坐在窗边，西湖就在眼前——苏堤、三潭印月、雷峰塔尽收眼底
- ⭐ **这是整个杭州性价比最高的"西湖景观位"**——网红餐厅花几百块才能看到的景，这里一杯茶搞定
