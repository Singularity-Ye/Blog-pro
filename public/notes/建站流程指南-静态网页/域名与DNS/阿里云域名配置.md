# 阿里云域名配置

> 购买域名并修改 DNS 服务器指向 Cloudflare。

## 步骤

1. 登录 [阿里云域名控制台](https://dc.console.aliyun.com/)
2. 搜索并购买域名（如 `example.com`）
3. 完成实名认证（国内必需）
4. 进入域名管理 → **DNS 修改**
5. 将 DNS 服务器改为 Cloudflare 提供的 Nameserver

## 修改 DNS 服务器

```
原 DNS 服务器：dns1.hichina.com / dns2.hichina.com
新 DNS 服务器：Cloudflare 提供的两个 Nameserver
```

## 等待生效

- 通常几分钟到 24 小时
- 可在 [whatsmydns.net](https://www.whatsmydns.net/) 检查全球解析状态
