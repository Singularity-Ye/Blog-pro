# LR 语法分析核心术语索引

本索引串联了自底向上语法分析（尤其是 LR0 和 SLR1）的关键概念。
## 1. 基础构造
- [[LR(0) 项目]]：语法分析的基本单位。
- [[核心项]] 与 [[闭包项]]：构成 DFA 状态的两种项目类型。
- [[增广文法]]：为了设置唯一的接受状态而引入。
## 2. 自动机理论
- [[NFA]] 与 [[DFA]]：通过子集构造法实现的转换。
- [[闭包运算]]：计算状态内所有等价项目的过程。
- [[跳转函数]]：状态间转换的逻辑驱动。
## 3. 分析表与决策
- [[Action 表]]：控制[[移进]]、[[归约]]、接受或报错。
- [[Goto 表]]：处理归约后的状态回溯。
- [[Follow 集合]]：[[SLR(1)]] 解决冲突的关键工具。

## 4. 冲突类型
- [[移进-归约冲突]]：当前状态既能移进又能归约。
- [[归约-归约冲突]]：当前状态存在多个可归约的产生式。
