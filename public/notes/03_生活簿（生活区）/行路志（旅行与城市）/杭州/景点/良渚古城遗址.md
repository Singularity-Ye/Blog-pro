# 良渚古城遗址

世界文化遗产（2019 年入选），距今约 5300-4300 年的新石器时代晚期古城遗址——比夏朝还早 1000 年。这里有中国最早的大型水利工程、最早的稻作农业遗址、最早的玉器文明。

## 基本信息

- **门票**：¥60（需提前预约，小程序"良渚古城"）
- **观光车**：¥20（⚠️ 园区约 14km²，步行不现实，**必须买**）
- **位置**：余杭区瓶窑镇
- **交通**：地铁 2 号线"良渚"站 B 出口 → 换乘 1222M 路公交（约 15min）→ 良渚古城遗址公园
- **开放时间**：9:00-17:00（16:00 停止入园）
- **游览时间**：建议 **4-5 小时**（半天）

## 核心看点

### 🏛️ 三大必看区域

| 区域 | 看点 | 建议时间 |
|------|------|:--:|
| **莫角山宫殿区** | 古城最高点，5000 年前的"紫禁城"——巨型夯土台基 | 40min |
| **反山王陵** | 出土大量玉琮、玉璧的高等级墓葬（复原展示） | 30min |
| **水利系统** | 中国最早的大型水利工程（世界最早的拦洪大坝） | 20min |

### 🏠 其他看点

| 看点 | 内容 | 特点 |
|------|------|------|
| 鹿苑 | 梅花鹿互动区 | 小朋友最爱，可以喂鹿 |
| 良渚博物院 | 免费，出土文物展示 | 玉琮王、玉璧王在此 |
| 稻作体验区 | 5000 年前的稻田复原 | 春天绿油油 |
| 良渚古城南城墙 | 保留的城墙遗址 | 感受 5000 年前的工程 |

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 莫角山台基 | 巨型夯土平台+蓝天 | 广角拍气势 |
| 鹿苑 | 和梅花鹿互动 | 蹲下拍，用树叶引鹿 |
| 稻田 | 大面积稻田+远山 | 低角度拍稻浪 |
| 城墙遗址 | 5000 年前的土墙 | 特写纹理 |

> 💡 **穿搭建议**：田园风/自然系——棉麻长裙+草帽，在稻田和鹿苑都很出片。

## 游玩攻略

### ⏰ 推荐路线（4h）

```
南入口 → 乘坐观光车 → 莫角山宫殿区（步行参观 40min）
    → 观光车 → 反山王陵+良渚博物院（40min）
    → 观光车 → 鹿苑（30min，小朋友必去）
    → 观光车 → 水利系统（20min）
    → 观光车 → 南城墙（20min）
    → 返回南入口
```

### 🍜 园内餐饮

- 园内有简餐/小卖部，但选择少且贵
- 🟢 **建议自带干粮/零食**，在稻田旁野餐
- 园外良渚镇有餐馆

## 五一提醒

- ⚠️ **必须提前预约**！五一名额有限，建议提前 3-5 天在小程序预约
- 🚗 自驾停车 ¥10/次，但五一可能满——建议地铁+公交
- 🧴 园区很大且户外为主，做好防晒+带水
- 👶 适合亲子（鹿苑+稻田+博物院）
