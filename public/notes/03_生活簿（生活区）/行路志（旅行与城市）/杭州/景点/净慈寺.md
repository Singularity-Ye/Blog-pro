# 净慈寺

西湖十景"南屏晚钟"所在地。与 [[雷峰塔]] 隔南山路相望，游客量却只有雷峰塔的十分之一——是南山路集群中最被低估的宝藏寺庙。

## 基本信息

- **门票**：¥10
- **位置**：南山路 56 号（雷峰塔马路对面）
- **开放时间**：6:30 - 17:00（16:45 停止售票）
- **游览时间**：30-60min
- **交通**：公交 4 路/12 路至净寺站

## 🍜 隐藏美食：素烧鹅

净慈寺**出口处**有一个不起眼的小窗口，售卖 **[[净慈寺素烧鹅]]**——酱汁卤制的多层豆腐皮卷，甜口，汁水四溢，约 ¥7/份。在杭州本地人心中地位极高。

> 💡 **最佳吃法**：买一盒素烧鹅，坐在南山路长椅上，看着雷峰塔吃——比花几百块去网红店香一万倍。
>
> 🔑 **不用买票进寺也能买！** 窗口在出口处（寺庙外侧），不买门票也能走到。

## 🟢 串联建议

```
方案 1（南山路经典）：
花港观鱼 → 净慈寺（30min，买素烧鹅）→ 步行 5min → 雷峰塔（1h）

方案 2（苏堤收尾）：
苏堤（2.8km）→ 苏堤南口 → 步行 5min → 净慈寺
    → 步行 5min → 雷峰塔 → 步行 8min → 太子湾公园

方案 3（深度人文）：
净慈寺（1h，敲钟+看济公殿）→ 雷峰塔 → 太子湾公园
    → 步行 15min → 浴鹄湾（秘境）
```

## ⚠️ 五一提醒

- 净慈寺游客量远少于雷峰塔——即使五一也不会拥挤
- 🟢 作为苏堤南口→雷峰塔之间的完美中转站
- 素烧鹅窗口不需要排队（不像灵隐寺素面）
- 🟢 **不用买票进寺也能买素烧鹅**——直接去出口处窗口
- 如果雷峰塔排队太久 → 先逛净慈寺，等塔人少了再去
- 寺内后山人极少——是五一"偷偷清净"的好去处
