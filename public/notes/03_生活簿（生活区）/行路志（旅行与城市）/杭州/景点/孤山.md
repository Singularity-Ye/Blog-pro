# 孤山

西湖中最大的天然岛屿。白堤连接北岸，西泠桥连接 [[苏堤]] 北口。山不高（38m），却是杭州人文密度最高的地方——西泠印社、浙江省博物馆、中山公园全在这巴掌大的岛上。

## 基本信息

- **门票**：免费
- **位置**：西湖中，白堤西端
- **高度**：38m（爬山 5 分钟）
- **游览时间**：1-2h（含西泠印社+省博）
- **交通**：白堤步行到达 / 公交至西泠桥站

## 孤山三宝

### 🏛️ 西泠印社

| 信息 | 说明 |
|------|------|
| 门票 | 免费 |
| 地位 | 中国最早的金石篆刻学术团体（1904 年创立）|
| 核心 | 山顶汉三老石室，珍藏东汉石碑 |
| 看点 | 四照阁（俯瞰西湖绝佳机位）、仰贤亭、华严经塔 |
| 时长 | 30-45min |

> 💡 西泠印社山顶的四照阁是整个孤山看西湖最好的位置——比平湖人少 10 倍，视野更开阔。

### 🏛️ 浙江省博物馆（孤山馆）

| 信息 | 说明 |
|------|------|
| 门票 | 免费（需预约）|
| 核心 | 陶瓷馆、漆器馆、雷峰塔地宫文物 |
| 镇馆之宝 | 雷峰塔地宫出土的鎏金铜佛 |
| 时长 | 30-60min |

### 🌳 中山公园

| 信息 | 说明 |
|------|------|
| 门票 | 免费 |
| 历史 | 清代行宫遗址（康熙/乾隆下江南驻跸处）|

## ⚠️ 五一提醒

- 孤山游客密度远低于白堤和苏堤——大部分人过白堤直奔苏堤，很少停留
- 🟢 是白堤徒步的完美"中场休息"
- 省博孤山馆五一人也不多（大家去之江新馆了）——但同样需要预约
- 孤山上没有商店——提前买水
- 山不高（38m），但台阶多，老人/小孩慢走即可
- 💡 最佳策略：在孤山停 1-1.5h，逛完三宝再走——比赶路有价值得多
