# 南宋御街

南宋时期的皇家御道，现为中山中路步行街段（约 1.5km）。与 [[河坊街]] 十字相交——比河坊街安静 70%，民国建筑+梧桐树荫更适合散步。是杭州老城区最有文艺气质的步行街。

## 基本信息

- **门票**：免费
- **位置**：**上城区** 中山中路（⚠️ 原图错误归入西湖区）
- **长度**：步行段约 1.5km（河坊街交汇处 → 鼓楼）
- **游览时间**：30-60min（散步拍照）
- **交通**：地铁 7 号线吴山广场站步行 3min

## 河坊街 × 南宋御街

两条街十字相交，风格完全不同：

```
河坊街（东西向，仿古商业街，热闹/游客/小吃）
   ×
南宋御街（南北向，中山路段，安静/本地/咖啡馆+书店）
```

| | [[河坊街]] | 南宋御街 |
|------|----------|---------|
| 风格 | 明清仿古 | 民国+南宋混搭 |
| 店铺 | 小吃+特产 | 咖啡馆+书店+独立设计 |
| 人流 | 🔴 五一人挤人 | 🟡 中等 |
| 适合 | 吃喝打卡 | 散步拍照 |
| 拍照 | ⭐⭐ | ⭐⭐⭐⭐ |

## 🟢 五一策略

河坊街打卡后立刻转入南宋御街——人少一大半：

```
河坊街（主街打卡 20min）→ 转入南宋御街（散步 40min）
                                ↓
                           鼓楼 → 中山南路
                                ↓
                           [[游埠豆浆]] / [[孙奶奶葱包烩]] / [[菊英面馆]]
```

## ⚠️ 五一提醒

- 南宋御街人比河坊街少——但交汇处还是人多
- 🟢 往鼓楼方向走，越走人越少
- 沿线咖啡馆五一座位紧张，可以外带边走边喝
- 地面南宋遗址在玻璃下——低头看，不要错过
- 💡 最佳串联：河坊街→南宋御街→鼓楼→中山南路美食→大马弄
