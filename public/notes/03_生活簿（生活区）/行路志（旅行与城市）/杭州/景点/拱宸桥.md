# 拱宸桥

京杭大运河最南端标志。杭州城区最高最长的石拱桥（长 98m，高 16m），始建于明崇祯四年（1631 年）。桥西历史街区免费开放——运河沿线最有韵味的区域。

## 基本信息

- **位置**：拱墅区，京杭大运河南端
- **门票**：免费（桥本身+桥西历史街区）
- **开放时间**：全天（桥西历史街区店铺约 10:00-22:00）
- **游览时间**：1-2h（含桥西历史街区）
- **交通**：
  - 🚢 水上巴士：武林门码头 → 拱宸桥，¥3，约 40min（推荐！）
  - 🚇 地铁：5 号线拱宸桥东站步行 10min

## 🌉 核心看点

### 拱宸桥本体

| 看点 | 说明 |
|------|------|
| 桥身 | 三孔薄墩联拱桥，明代古桥 |
| 桥顶 | 站桥顶看运河两端——南望武林门，北望运河天地 |
| 石狮 | 桥栏石狮雕刻 |
| 夜景 | 桥上亮灯后倒映运河，绝美 |

### 桥西历史街区

| 区域 | 看点 | 推荐 |
|------|------|:--:|
| 运河博物馆群 | 伞博物馆+刀剪剑博物馆+扇博物馆 | ⭐⭐⭐⭐⭐ 全免费 |
| 桥西直街 | 清末民初民居+茶馆+文创店 | ⭐⭐⭐⭐ |
| 邻里长桌宴 | 五一场活动 | ⭐⭐⭐ |
| 露天茶室 | 河边喝茶看运河 | ⭐⭐⭐⭐ |

> 💡 **三个博物馆全免费**——伞/刀剪剑/扇，每个 30min 就能逛完。雨天备选首选。

## ⚠️ 五一提醒

- 水上巴士到拱宸桥是最推荐的交通方式——**花 ¥3 看 40min 运河风光**
- 五一人比西湖少很多，但桥上拍照的人还是多
- 🟢 建议 **10:00 前到**或**16:00 后到**（避开 11:00-15:00 高峰）
- 博物馆群免费但五一可能限流——如果排队就先逛桥西直街
- 🚫 至 [[武林夜市]] 约 6.9km，**不可步行**，需打车 25-35min 或地铁约 30min
