# 法净寺贰捌玖素食馆

[[法净寺]]（中天竺）门口的素食馆。什锦面是绝对主角——番茄汤底，十多种配料（豆腐干、油球、腐竹、青豆、菌菇、腰果），每一筷子下去跟挖宝似的。在灵隐景区三家素斋中，**性价比最高**。

## 推荐

| 菜品 | 价格 | 笔记 |
|------|:--:|------|
| **什锦面** | ¥39 | 招牌，十多种料，番茄汤底 |
| 卤豆腐干 | ¥8 | 咸香入味，泡面汤吃 |

- 🕐 7:00 - 21:00
- 💰 什锦面 ¥39
- 🚶 从灵隐寺步行 15min / 从法喜寺步行 10min

## 🟢 串联

天竺三寺都在一条路上：

```
灵隐寺 → 步行 15min → 法镜寺 → 步行 5min → 法净寺
    → 贰捌玖素食馆（午餐 ¥39）→ 步行 5min → 法喜寺（¥10，网红机位）
```

## ⚠️ 五一提醒

- 贰捌玖不像十方苑那么出名——五一人不多
- 🟢 如果十方苑排队长 → 这里是完美替代（步行 15min）
- 7:00 就开门——灵隐寺逛完正好来吃
- 什锦面量很大——一个人可能吃不完
- 天竺路上法净寺/法镜寺/法喜寺都可以一起逛
