# 孙奶奶葱包烩

做了三十多年的杭州经典小吃。登上过央视。春饼包油条+葱，铁板压至酥脆，刷甜辣酱——老杭州的童年记忆。90 岁奶奶的祖传手艺。

## 基本信息

- **交通**：地铁 5 号线江城路站步行 8min

## 什么是葱包烩

南宋流传至今的小吃。传说百姓痛恨秦桧害死岳飞，把面饼当作"秦桧夫妇"，裹油条和葱，铁板压扁烤熟吃掉——"烩"谐音"桧"。吃了九百年，恨意早已消散，只剩满口酥香。

| 工序 | 说明 |
|------|------|
| 1. 摊春饼 | 薄面皮铺开 |
| 2. 放油条 | 现炸油条折断放上 |
| 3. 加葱 | 新鲜小葱 |
| 4. 铁板压 | 用铲子压扁压酥 |
| 5. 刷酱 | 甜面酱+辣酱 |
