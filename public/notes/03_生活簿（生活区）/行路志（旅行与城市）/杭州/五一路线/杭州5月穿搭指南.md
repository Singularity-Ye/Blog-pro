# 杭州 5 月穿搭指南

5 月的杭州气温在 **18-30°C**，早晚温差大，加上梅雨前奏，穿搭需要兼顾防晒、防雨、舒适三重需求。

## 气温概况

| 时段 | 温度 | 体感 |
|------|------|------|
| 早晨 6-8 点 | 18-22°C | 微凉，需要薄外套 |
| 上午 9-12 点 | 24-28°C | 舒适，短袖即可 |
| 下午 13-16 点 | 28-32°C | 热，注意防晒 |
| 傍晚 17-19 点 | 24-26°C | 舒适 |
| 夜间 20-23 点 | 20-22°C | 微凉 |

## 穿搭建议

### 👕 衣服
- **日间**：短袖 / 薄长袖 / 防晒衫
- **早晚**：薄外套 / 防风衣（必带！早起去[[断桥残雪]]会冷）
- **雨天**：防水外套 > 雨伞（西湖边风大，伞不好撑）

### 👟 鞋子（最重要！）
- ✅ **舒适的运动鞋 / 徒步鞋** — 一天 2 万步起
- ❌ 不要穿新鞋（磨脚）
- ❌ 不要穿高跟鞋 / 凉鞋（九溪/龙井的路不好走）
- 💡 可以带一双拖鞋备用（下雨鞋湿了能换）

### 🎒 配件

| 必带 | 建议 |
|------|------|
| 🧢 遮阳帽 | 西湖边没什么树荫 |
| 🕶️ 墨镜 | 拍照也好看 |
| 🔋 充电宝 | 手机导航+拍照耗电极快 |
| 🧴 防晒霜 | SPF50+，每 2 小时补涂 |
| 🦟 驱蚊水 | [[西溪湿地]]/[[九溪十八涧]] 蚊子很凶 |
| 💧 水壶 | 景区内水 ¥5-10/瓶，自带省钱 |
| 💰 少量现金 | 部分小吃摊/寺庙只收现金 |

## ⛅ 天气预案

| 天气 | 应对 |
|------|------|
| ☀️ 晴天 | 帽子+墨镜+防晒霜，走[[西湖秘境线]]/[[九溪十八涧]] |
| 🌦️ 小雨 | 穿防水外套继续走，西湖雨景更美，人也更少 |
| 🌧️ 大雨 | 改室内行程：[[杭州博物馆]] / [[南宋德寿宫遗址博物馆]] / [[03_生活簿（生活区）/行路志（旅行与城市）/杭州/活动/CP32漫展]] / 商场 |

## 五一特别提醒

- 5 月杭州偶有阵雨，**随身带把折叠伞**
- 西湖边风大，裙子容易被吹起来（穿裙子的注意）
- 爬 [[北高峰]] / [[十里琅珰]] 会出汗，带一条毛巾
