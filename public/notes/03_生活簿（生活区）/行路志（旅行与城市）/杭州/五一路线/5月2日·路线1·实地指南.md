## 🚇 Step by Step

### 第一程：下沙 → 杨公堤

```
7:30  生活三区出门
        ↓ 🚶步行 10min
7:40  文泽路站 A口进
        ↓ 🚇1号线（往湘湖方向）
        经：高沙路→金沙湖→客运中心→火车东站→武林广场→凤起路
        ↓ ⚠️不要在龙翔桥下车！
8:40  定安路站 出站
        ↓ 🚴扫共享单车（南山路方向骑行 15min）
        或 🚌公交 4路/12路/31路 → 苏堤站
9:00  杨公堤·郭庄 到达
```

### 第二程：西湖秘境线

```
9:00  🏯 郭庄（¥10）
        └── 进门左手临湖茶室，点一杯龙井（~¥40）
        └── 30min，拍照+喝茶
        └── 📸 机位：茶室窗边（框景西湖）

9:45  🚶沿杨公堤往南步行 10min
        ↓
      🐢 乌龟潭
        └── 于谦祠码头木栈道
        └── 📸 机位：舒云亭边桥上等摇橹船穿桥洞
        └── 20min，发呆+拍照

10:30 🚶沿水边小路往南 10min
        ↓
      🏚️ 浴鹄湾
        └── 霁虹桥（茅草屋+水面倒影）
        └── 📸 机位：桥对面拍全景；武状元坊用桥洞做框景
        └── 30min，拍照+发呆

11:15 🚶往南 → 花港观鱼
        └── 看两眼红鲤鱼，5min 够了
```

### 第三程：南山路

```
11:30 🍜 净慈寺素烧鹅
        └── 净慈寺进门左手边小窗口
        └── 素烧鹅 ¥8/条 + 豆腐干 ¥3/串
        └── ⚠️ 中午前到还能买到，再晚可能卖完
        └── 买完坐在对面南山路长椅上吃

12:00 🏯 净慈寺（¥10）
        └── 南屏晚钟 ¥10/次
        └── 后山远眺雷峰塔+西湖
        └── 30min

13:00 🍜 素春斋
        └── 净慈寺出口外几十米
        └── 单人套餐 ¥68（素什锦面+3小菜）
        └── ⚠️ 周四休息，今天周日✅
```

### 第四程：啤酒节 + 雷峰塔日落

```
14:00 🍺 中国美院南山校区 啤酒节
        └── 南山路步行往北 10min
        └── 精酿啤酒+手作面包+国美20周年展
        └── 🆕 今天第一天开幕！
        └── 逛 1.5-2h，喝两杯精酿（¥50 左右）

16:00 🍵 空档／逛南山路
        └── 可去 [[青藤茶馆]]（¥200，南山路店，喝茶等日落）
        └── 或直接去雷峰塔

16:30 🗼 雷峰塔（¥40）
        └── ⚠️ 进塔前先路过 [[福缘居]] 取号！
        └── 登塔（有电梯），先看 2-4 层
        └── 5 层最高观景台
        └── 🆕 五一开到 20:00

18:00 🌇 塔顶看日落
        └── 这个时间上去正好
        └── 📸 苏堤+三潭印月+湖心亭全景

18:30 🍜 福缘居
        └── 脆皮大肠 ¥68 + 油淋鸡 ¥58
        └── 如果之前忘了取号→现在去可能要排 1h+
        └── 🟢 备选：[[新丰小吃]]（如果排队太久）
```

### 第五程：回下沙

```
20:00 🚇 定安路站 进站
        └── ⚠️ 不要在龙翔桥上！限流可能进不去
        └── 🚇1号线（往下沙江滨方向）
        └── 经：凤起路→武林广场→火车东站→客运中心→文泽路
20:45 🏠 文泽路站 出站
        └── 🚶步行 10min 回生活三区
```

## 🚨 应变决策树

### 下雨了怎么办？

| 雨量 | 方案 |
|------|------|
| 🌧️ 小雨 | 照走。杨公堤梧桐全覆盖，基本淋不到。石板路注意防滑。 |
| 🌧️ 中雨 | 砍掉秘境线前半段。直接去南山路：净慈寺→素春斋→美院啤酒节（室内）→雷峰塔（室内） |
| 🌧️ 大雨 | 纯室内方案：🚇1→武林广场→换3→古墩路→[[03_生活簿（生活区）/行路志（旅行与城市）/杭州/活动/天目里春日艺术装置展]]（茑屋书店+%Arabica+买手店，全室内）。或直接去 CP32漫展。 |

### 某个节点出问题了？

| 问题              | 方案                                        |
| --------------- | ----------------------------------------- |
| 郭庄排队太长          | 跳过。直接去乌龟潭——免费且没人。                         |
| 素烧鹅卖完了          | 没事，素春斋也有素烧鹅 ¥8，门口档口也有。                    |
| 素春斋排队           | 去 [[净慈寺素烧鹅]] 加豆腐干先垫着，或南山路找家面馆。            |
| 美院啤酒节人太多        | 去 [[青藤茶馆]]（南山路店）喝茶休息。或直接去雷峰塔。             |
| 雷峰塔排队太长（>30min） | 不登塔。塔前广场拍外观也很出片。转去太子湾公园散步。                |
| 福缘居排队 >1h       | 🟢 转 [[新丰小吃]]（附近有分店，¥15 快速搞定）。或南山路随便找家面馆。 |
| 太累了走不动          | 雷峰塔之后直接回。后面福缘居可改天去。                       |
| 回程龙翔桥站限流        | 这是为什么我让你在**定安路**上下车！定安路比龙翔桥人少得多。          |


### CP32 突然有票了想去？

```
但会错过啤酒节和雷峰塔日落——两者只能选一个。
```
