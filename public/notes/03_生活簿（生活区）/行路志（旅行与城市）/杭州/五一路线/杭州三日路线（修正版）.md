# 杭州三日路线（修正版）

> 基于路线拓扑图分析结果，逐条修正原始错误后的三日行程规划。

---

## Day 1：西湖环湖 + 南山文化 + 老城夜市

**主题**：经典西湖 + 步行友好集群

> 🔴 **五一骑行禁令**：苏堤/白堤/断桥全面禁行共享单车和非机动车。Day 1 全程步行，穿舒适的鞋。

```
上午  ［断桥残雪］→ 白堤 → 孤山（西泠印社）→ 岳王庙（25元）
中午  ［苏堤春晓］→ 花港观鱼
下午  ［苏堤南口］→[步行5分]→ 净慈寺（10元）→[步行5分]→ 雷峰塔（40元）
       ↓
      ［太子湾公园］（免费）
傍晚  苏堤南口 →[步行5分]→ 河坊街 → 南宋御街
       ↓
       杭州博物馆（如有预约）
晚餐  ［河坊街夜市］或 吴山广场
```

| 段 | 交通 | 耗时 | 备注 |
|----|------|------|------|
| 断桥 → 岳王庙 | 步行 | 15min | 白堤沿线 |
| 岳王庙 → 苏堤 | 步行 | 20min | 🔴 五一禁行单车，只能步行 |
| 苏堤南口 → 净慈寺 | 步行 | 5min | ✅ 正确起点 |
| 净慈寺 → 雷峰塔 | 步行 | 5min | ✅ |
| 雷峰塔 → 太子湾 | 步行 | 8min | ✅ |
| 苏堤南口 → 河坊街 | 步行 | 5min | ✅ |
| 河坊街 → 杭博 | 步行 | 5min | ✅ 在同一座山 |

> 🟢 **苏堤挤爆时的替代方案**：从苏堤南口拐进杨公堤 → [[浴鹄湾]]/乌龟潭（免费秘境）→ [[郭庄]]（10元，江南园林喝茶看湖）→ 曲院风荷。这条"西湖秘境线"避开 99% 的游客。

---

## Day 2：灵隐佛国 + 龙井九溪

**主题**：寺庙祈福 + 茶山徒步

> 🆓 **免票政策**：飞来峰+灵隐寺自 2025.12 起**免费**！必须「杭州灵隐飞来峰」小程序提前预约。五一限额 6.5 万/天。
>
> 🔴 **预约**：提前 3-7 天在小程序抢预约。提前 1 天 = 已约满。
>
> 🔴 **到达时间**：五一不是"8 点前到"，是 **6:30 前到**。

```
上午  ［飞来峰景区大门］→[🆓免费需预约]→ 飞来峰石窟造像
       ↓
      灵隐寺 → 大雄宝殿 / 药师殿 / 华严殿
中午  ［法喜寺］（门票10元，素斋5元）
下午  龙井村 → 九溪十八涧
```

| 段 | 交通 | 耗时 | 备注 |
|----|------|------|------|
| 市区 → 飞来峰 | 🚗 打车 | 20-30min | 🔴 6:00 出发，6:30 到门口排队。**地铁不到灵隐！** |
| 飞来峰大门 → 灵隐寺 | 步行 | 10min | 先在飞来峰游览 |
| 灵隐 → 法喜寺 | 步行 | 20min | 天竺路步行道 |
| 法喜寺 → 龙井村 | 打车 | 15min | 或步行 40min |
| 龙井村 → 九溪 | 步行 | 1.5-2h | 🔴 九溪路极窄，必须从公交站徒步进，勿打车 |

---

## Day 3：运河文化线 + 武林夜市

**主题**：京杭大运河 + 博物馆群 + 香积寺

> ⚠️ **关键纠正**：拱宸桥到武林夜市约 6.9km，不是步行 15 分钟！需打车或地铁。
>
> 🆕 **灵隐平替**：[[香积寺]]（20元）是国内唯一供奉食神的寺庙，五一清净且素面极佳。如果昨天被灵隐的人潮劝退，这里是最好的疗愈。

```
上午  ［武林门码头］→[水上巴士3元/40min]→ 信义坊码头 →[步行10min]→ 香积寺（20元）
       ↓
       吃素面，感受禅意（灵隐的拥挤这里完全没有）
中午  ［拱宸桥］→ 桥西历史街区
下午  ［运河博物馆群］：伞博物馆 / 刀剪剑博物馆 / 扇博物馆（均免费）
       ↓
      小河直街 → 大兜路历史街区
傍晚  小河直街 →[🚗打车25-35min 或 🚇地铁30min]→ 武林夜市
```

| 段 | 交通 | 耗时 | 备注 |
|----|------|------|------|
| 武林门 → 信义坊 | 水上巴士 | 30min | 3元，沿途运河风光 |
| 信义坊 → 香积寺 | 步行 | 10min | ✅ |
| 香积寺 → 拱宸桥 | 步行/骑行 | 15min | 沿运河 |
| 拱宸桥 → 小河直街 | 骑行 | 12min | ✅ 沿运河 |
| 小河直街 → 武林夜市 | 🚗 打车 | 25-35min | ❌ 原图说步行15分是错的 |
| 小河直街 → 武林夜市 | 🚇 地铁 | ~30min | 备选 |

## ⚠️ 五一特别注意事项

1. **灵隐预约**：提前 **3-7 天**在"西湖旅游"小程序抢票，提前 1 天 = 已售罄
2. **断桥/灵隐到达时间**：不是"8 点前"，是 **6:30 前**。8 点已是旅行团天下
3. **五一骑行禁令**：苏堤/白堤/断桥核心区**全面禁行**共享单车和非机动车，全程步行
4. **虎跑路拥堵**：去动物园/宋城的时间 ×3~4 倍
5. **预约必须提前**：杭州博物馆、德寿宫、良渚古城均需提前在公众号预约
6. **灵隐寺免费需预约**：2025.12起免票！「杭州灵隐飞来峰」小程序预约，五一限额 6.5 万
7. **灵隐寺平替**：[[香积寺]]（拱墅区，20元）供奉食神，五一清净，素面极佳
8. **九溪交通**：九溪路极窄，必须从九溪公交站徒步进，勿打车
9. **河坊街美食**：别在主街买"轰炸大鱿鱼"，拐进[[大马弄美食指南|大马弄]]吃正宗葱包烩、油墩儿
10. **起点模糊陷阱**："西湖"不是精确起点，必须确认具体位置（断桥/苏堤南口/湖滨）
11. **水上巴士**：3 元一趟，但排队人多，建议早到武林门码头
12. **灯光秀**：[[钱江新城灯光秀]]五一可能每日加场，关注"杭州城管"公众号
