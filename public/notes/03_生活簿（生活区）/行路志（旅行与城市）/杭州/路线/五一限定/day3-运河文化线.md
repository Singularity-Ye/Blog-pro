# Day 3 运河文化线 + 武林夜市

> 京杭大运河 + 博物馆群 + 香积寺。全程没有旅行团——这才是本地人逛的杭州。

## Route Points

1. [[武林门码头]]
2. [[拱宸桥]]
3. [[小河直街]]
4. [[大兜路历史街区]]
5. [[香积寺]]
6. [[武林夜市]]

## Schedule

- 09:00 [[武林门码头]] — 水上巴士 ¥3，40min 运河风光
- 09:40 [[拱宸桥]] — 大运河最南端标志
- 10:00 运河博物馆群 — 伞/刀剪剑/扇博物馆，均免费
- 12:30 [[小河直街]] — 步行 20min，宋式美学市集
- 15:00 [[大兜路历史街区]] — 文创+轻食
- 16:45 [[香积寺]] — ¥20，国内唯一供奉食神的寺庙
- 18:15 [[武林夜市]] — 打车 25-35min 或地铁 30min

## Route Summary

京杭大运河深度一日游。水上巴士→拱宸桥→博物馆群→小河直街→香积寺→武林夜市。全程运河沿线，节奏极慢。香积寺是灵隐寺的最佳平替——五一清净且素面极佳。

## 决策树

```
Q: 水上巴士排队长？
  → 建议 9:00 前到码头
  → 等不了 → 改地铁到拱宸桥东站（1号线 25min）

Q: 香积寺素面卖完了？
  → 去胜利河提前吃晚饭

Q: 下雨了？
  → 博物馆群室内多逛 1-2 个
  → 雨中运河反而更有意境
```

## Notes

- 🚢 水上巴士 ¥3/趟，排队人多建议早到
- ⚠️ 拱宸桥到武林夜市约 6.9km，不可步行，需打车或地铁
- 🆕 香积寺（¥20）是国内唯一供奉食神的寺庙，灵隐平替
