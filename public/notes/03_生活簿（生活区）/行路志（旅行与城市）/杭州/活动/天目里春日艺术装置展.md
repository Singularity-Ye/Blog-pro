# 天目里·春日艺术装置展

天目里园区——意大利建筑大师伦佐·皮亚诺（蓬皮杜中心设计者）在中国的首个作品。春日艺术装置散布整个园区 + 青竺微物可观市集（至 5/5）。

## 基本信息

- 📅 艺术装置展：至 **5/31**（整个五月都在）
- 📅 青竺微物可观市集：至 5/5
- 📍 天目里园区（西湖区天目山路）
- 💰 免费
- 🚇 地铁 3 号线「古墩路」站步行 5 分钟

## 园区必逛

| 地点 | 笔记 |
|------|------|
| 📖 **茑屋书店** | 中国大陆首店，选书品味一流 |
| ☕ **% Arabica** | 京都网红咖啡，拍照打卡圣地 |
| 🛍️ 买手店 | 独立设计师品牌集合 |
| 🎨 户外装置 | 散落园区的春日主题艺术装置 |
| 🍽️ 餐饮 | 多家中高档餐厅和咖啡馆 |

## 🟢 适合什么场景

- 🌧️ **下雨天** —— 室内外结合，不受天气影响
- 📸 **拍照** —— 建筑本身就是艺术品
- ☕ **文艺休息日** —— 西湖暴走后的慢节奏一天
- 🛍️ **购物** —— 茑屋书店+买手店可以逛一下午
