# 卷轴弹窗：多场景统一目录展开器

所有非单篇物件最终进入同一个弹窗系统，但背景随场景而变：

```javascript
$bgSrc={
  sceneMode === 'travel'    ? BLOG_NEW_ASSETS.oceanMap
  : sceneMode === 'archive' ? BLOG_NEW_ASSETS.archiveIndexBg
  : BLOG_NEW_ASSETS.scrollOpen
}
```

林间卷轴用竹简/玉卷背景，旅图案台用海图样式，旧札柜用抽屉索引样式，普通场景用打开的羊皮纸卷轴。同一个数据列表，在不同场景穿不同的衣服。

弹窗有两种模式：普通列表（`modalNotes`）和搜索模式（`archiveMode === 'search'`）。搜索模式从 `notesData` 收集最多 24 个展示标签，支持标题和标签匹配——旧札柜不仅是视觉装饰，还是一个可用的本地笔记检索入口。

> 一个弹窗，四种皮肤，两种模式。数据来自同一份索引，但读者在旅案上翻目录和在旧柜里翻抽屉的感受完全不同。
