# 松果星寰仪：首页卡片升级设计总结

## 一、总体设计原则

这次卡片升级的核心变化，是从“标准卡片背景”升级为“世界观内的器物”。

旧版卡片更像普通 UI 素材：
只是横向底板 + 不同材质 + 少量纹理。

新版卡片则强调：
每张卡片都应该像对应大陆自然生长出来的形状。

因此，卡片不再受矩形、圆角、叶片这些固定模板限制，而是让主题决定外轮廓，让边框本身参与叙事。

### 统一原则
* **轮廓自由化**：卡片不必是规整牌匾，而可以是卷轴、水湾、星盘、冰械、花笺等自然延展的形态。
* **边框叙事化**：边框不是装饰边，而是主题信息的主要承载区。例如：松窗的卷轴、听雨的纸舟、星络的星盘、玄枢的齿轮、浮生的桃花与命轨。
* **中心留白稳定**：虽然边缘元素丰富，但中心区域必须保持干净，方便放标题 and 简介文字。
* **整体亮度偏明亮**：首页背景是浅色梦幻风，卡片不能太暗，否则会变成脏斑。
* **每张有主符号**：每张卡片都必须有一个一眼能识别的视觉锚点，而不是只换颜色。

## 二、五张卡片设计思路

### 1. 听雨寄笺坞：水湾信笺牌
* **主题定位**：对应留言 / 联系页面。
* **核心气质**：江南烟雨、折纸投信、水岸灯火、人间温度。这张卡片的目标不是做成普通信纸，而是把“信笺”和“水湾”融合在一起。
* **主要元素**：水波形外轮廓、折角信笺、青玉色水纹、竹影、木栈桥、纸舟、莲灯、暖色灯笼、雨后晨雾。
* **创意点**：这张最重要的创意，是把“投信”从静态纸张变成了水上漂流的过程。它不是一封躺在桌上的信，而像是一张被水汽托起的信笺：水湾托住纸舟，纸舟载着留言，暖灯引导它抵达屋主。
* **风格关键词**：江南、烟雨、青玉水、纸舟、莲灯、晨雾、温柔、有人间烟火气。

### 2. 松窗灵笈台：卷轴书院牌
* **主题定位**：对应文章 / 笔记 / 书卷入口。
* **核心气质**：松窗灯火、灵笈藏书、山谷书院、文气与仙气并存。新版不再使用单纯叶片形，而是让卡片像一张被灵气撑开的卷轴，边缘自由舒展。
* **主要元素**：自由卷轴轮廓、松枝、竹叶、暖色灯笼、山谷溪流、木质书阁、竹简、卷轴、玉简架、石碑、金色描边、远山薄雾。
* **创意点**：这张最精彩的地方，是外轮廓完全不被固定卡片形束缚。它像一幅正在展开的灵笈卷轴，松枝和灯笼从边缘自然长出，右侧书阁则让卡片有了“藏书深谷”的具体场景感。它已经不是“森林风卡片”，而是一张打开后能看见松窗书院的法器卷轴。
* **风格关键词**：松风、竹简、书阁、灵笈、暖灯、溪流、卷轴、清雅、书卷气。

### 3. 星络天机原：星沙天机牌
* **主题定位**：对应图谱 / 知识关联 / 思维星图。
* **核心气质**：星沙漫天，天机引线，因果节点，荒漠星盘。这张不能只是普通羊皮纸地图，而要表现“星络”和“天机”。因此它把沙漠残卷、星盘、轨道线、天文仪器融合成一张星沙导航牌。
* **主要元素**：沙金色不规则残卷轮廓、星盘、罗盘、星座连线、金色节点、轨道弧线、沙丘纹理、水晶矿簇、天文仪挂坠、星沙颗粒、少量紫色宝石点缀。
* **创意点**：它的边框像一块从星沙荒原中挖出的天机残片。左侧是星盘，右侧是晶体与阵图，下方是沙丘与轨迹线。这让它既有沙漠的地貌感，又有“知识节点互相连接”的抽象意味。这张卡片特别适合作为图谱入口，因为星线和节点天然对应 Obsidian 的关系网络。
* **风格关键词**：星沙、因果线、星盘、罗盘、沙金、阵图、知识节点、天机残卷。

### 4. 玄枢造物坊：冰械炼金牌
* **主题定位**：对应项目 / WebGL / 三维法宝 / 技术实验。
* **核心气质**：雪峰玄枢，代码淬火，冰晶机械，蒸汽炼金。它不能只是冰雪卡片，而要体现“造物坊”的机械感。因此新版加入了冰晶、雪峰、齿轮、蒸汽、蓝色能量与黄铜结构。
* **主要元素**：冰晶外轮廓、雪山峰线、黄铜齿轮、管道、蒸汽、蓝色能量核心、炼金阵线、机械仪器、冰蓝水晶、金属框架、冷雾。
* **创意点**：这张的关键是“冷暖对撞”：外层是冰雪、雪峰、冷蓝水晶；内部是黄铜、齿轮、机械、炉心能量。它形成了一种“冰山之中藏着机械心脏”的感觉。和玄枢造物坊大陆本身的设定一致：自然雪峰被工程与炼金术改造成造物机关。
* **风格关键词**：雪峰、冰晶、黄铜、齿轮、蒸汽、炼金、蓝色能量、机械法器、冷暖对比。

### 5. 潮汐湾：海流归航牌
* **主题定位**：对应重置视角 / 回归主控大陆。
* **核心气质**：潮汐、归航、海风、清澈、重回主星。这张不能只是旧版的圆润水滴牌，而应该更像一片被海流雕刻出的潮汐器物。
* **主要元素**：自由海浪轮廓、海泡、贝壳、海星、码头、海图线、罗盘、气泡、珍珠、浅蓝水流、金色潮线、白色浪花。
* **创意点**：这张卡片的外形像一块由海浪、贝壳 and 罗盘共同构成的海流牌。左下角的小码头暗示“归航”，中间的留白像平静海面，边缘翻卷的浪花则形成流动感。它作为“潮汐湾 / 重置视角”非常合适，因为潮汐天然有“回归、复位、循环”的含义。
* **风格关键词**：海浪、潮汐、归航、贝壳、罗盘、码头、泡沫、海图、清透、重置。

### 6. 浮生道迹屿：花瓣命笺牌
* **主题定位**：对应关于本人 / 个人履历 / 自我介绍。
* **核心气质**：浮生若梦，道迹留痕，桃花照影，命轨回望。这张是整套升级的灵感来源之一。它成功的原因是完全脱离了标准卡片形状，而像一张自然舒展的花瓣云笺。
* **主要元素**：花瓣云笺轮廓、淡粉紫渐变、桃花枝、漂浮花瓣、镜湖水纹、三生石 / 镜石意象、金色命轨虚线、星盘 / 命盘纹理、云雾边缘、柔光。
* **创意点**：这张最重要的不是“花瓣好看”，而是它把个人页做成了“本命牌”。它没有强调功能，而是强调人格气质：经历、履历、兴趣、情感、成长轨迹，都被抽象成花瓣、镜湖和命轨。它是整套卡片中最柔软、最私人、最有“自我映照感”的一张。
* **风格关键词**：浮生、桃花、命轨、镜湖、三生石、云笺、柔光、梦幻、本命感。

## 三、整套卡片的风格层级

新版卡片可以理解成六种不同的“界面灵物”：

| 卡片 | 主要气质 | 主符号 |
| :--- | :--- | :--- |
| **松窗灵笈台** | 清雅书卷 | 卷轴、书阁、松枝 |
| **星络天机原** | 神秘理性 | 星盘、星线、沙金 |
| **玄枢造物坊** | 冷冽工学 | 冰晶、齿轮、蒸汽 |
| **听雨寄笺坞** | 江南人情 | 纸舟、水湾、灯火 |
| **浮生道迹屿** | 个人命格 | 桃花、命轨、镜湖 |
| **潮汐湾** | 回归复位 | 海浪、罗盘、码头 |

## 四、这轮升级的关键经验

### 1. 不要让 UI 只是 UI
如果网站本身是幻想世界观，那么 UI 组件也应该是世界观的一部分。卡片可以不是“按钮背景”，而是：一张卷轴、一枚星盘、一块冰晶机械铭牌、一封水上信笺、一枚本命花笺、一片潮汐海图。

### 2. 外轮廓比内部纹理更重要
这次最成功的地方，就是外轮廓变得自由了。用户第一眼看到的不是细节，而是整体形状。因此卡片设计优先级应该是：**轮廓 > 主色 > 主符号 > 边缘细节 > 中心纹理**。

### 3. 边缘负责叙事，中心负责阅读
这些卡片最终要放标题和简介，所以不能把中间画满。最好的方式是让边缘讲故事，中间保持呼吸感。

### 4. 每张卡片都应该有“性格”
旧版卡片像同一套模板的不同皮肤。新版卡片则像六个不同门派的信物：书卷派、星图派、机关派、水信派、浮生命牌、潮汐归航牌。这样首页导航才有真正的世界观沉浸感。
