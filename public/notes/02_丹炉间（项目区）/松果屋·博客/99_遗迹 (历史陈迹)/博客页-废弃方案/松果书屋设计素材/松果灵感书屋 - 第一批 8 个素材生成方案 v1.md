# 一、先定统一风格基准

在生成单个素材前，最好先统一这批素材的“世界观语言”。

## 整体风格关键词

你这套博客页建议统一成：

- **手绘插画感**
- **童话式剖面空间**
- **温暖安静的小型私人书屋**
- **略带绘本气质**
- **有一点 Obsidian / 旅行 / 世界观设定的个人气息**
- **柔和，不要过度写实**
- **不要太扁平，也不要太厚重 3D**
- **适合做网页分层素材**

## 视觉关键词

- 松果外壳
- 剖面小屋
- 暖黄灯光
- 木质书架
- 米白旧纸
- 浅绿色青蛙
- 小而精的图书馆陈设
- 旅行地图 / 卷轴 / 笔记本 / 技术机关书

## 配色建议

- 背景环境：雾蓝、浅灰蓝、夜蓝
- 松果：棕黄、焦糖棕、深褐
- 灯光：暖黄
- 青蛙：浅绿、嫩绿
- 书页：米白、旧纸黄
- 卷轴：羊皮色
- 技术书：深蓝 / 深紫 + 少量发光点缀
- 点缀色：火漆红、路线蓝、标签粉

# 二、第一批 8 个素材清单

## 1）松果书屋主背景

### 作用

整个博客页的主视觉舞台。

### 目标

生成一个**松果剖面书屋的大背景**，作为页面主体空间。  
注意：这不是完整网页截图，而是一个可以当作背景的大场景插画。

### 画面内容建议

- 一个大型松果剖面
- 内部是温暖的小书屋空间
- 有书架、小平台、小桌面、灯
- 内部预留一些空位，方便后续摆放交互物件
- 不要让元素太满，留出“放项目入口”的位置
- 画面偏居中或略偏右都可以

### 输出建议

- **横向**
- 比例：**16:9**
- 最好两版：
    1. **完整背景版**
    2. **偏空场景版**（留更多空位）

### 是否透明背景

- **否**
- 这是大背景

### 生成提示词草案

你可以喂给 Image2 这样的方向：

> A whimsical hand-drawn illustration of a giant pinecone cross-section transformed into a cozy miniature library. The pinecone shell is warm brown and textured, with a cutaway interior showing a tiny reading room. Inside are small wooden bookshelves, little platforms, a reading nook, warm hanging lamps, paper books, scrolls, and a quiet personal study atmosphere. The scene should feel like a fairy-tale private study, warm, intimate, gentle, and imaginative. Leave several open spaces for placing interactive objects later. Soft painterly shading, slightly storybook-like, warm yellow light, soft outlines, calm blue surrounding background, suitable as a website main background. Not a full webpage screenshot, but a clean standalone environment illustration.

### 备注

这是最重要的一张，建议你后面可以单独多迭代几次。

## 2）青蛙管理员（躺着读书版）

### 作用

作为博客页中的“馆长 / 居民 / 吉祥物”。

### 目标

做一个最核心的青蛙角色姿态，延续你现有那张“青蛙躺在松果屋里看书”的感觉。

### 画面内容建议

- 小青蛙躺着
- 手里捧一本小书
- 表情放松、安静、微微呆萌
- 带一点“长期住在书屋里的管理员”气质
- 可以有一点淡淡腮红
- 风格要尽量贴近你现有参考图的温暖手绘感

### 输出建议

- 比例：**1:1** 或 **4:5**
- 单独角色素材

### 是否透明背景

- **是，最好透明背景**
- 方便前端摆放到场景里

### 生成提示词草案

> A cute small frog character in a cozy hand-drawn storybook style, lying down while reading a small book. The frog should feel like a quiet library resident or little inspiration keeper. Soft pastel green body, gentle facial expression, slightly sleepy and peaceful mood, tiny rosy cheeks, whimsical and warm. Clean silhouette, simple but expressive pose, soft painterly shading, suitable as a standalone website character asset with transparent background.

### 备注

这张最好作为整个角色风格锚点。

## 3）青蛙管理员（抱地图册版）

### 作用

用于强化“旅行地图册”那一类项目入口的氛围。

### 目标

做第二个常用姿态：青蛙抱着一本地图册或旅行手册。

### 画面内容建议

- 青蛙坐着或半靠着
- 怀里抱一本地图册
- 地图册上可以隐约有路线、折页、标记
- 表情像在认真研究下一段旅程

### 输出建议

- 比例：**1:1** 或 **4:5**
- 单独角色素材

### 是否透明背景

- **是，透明背景**

### 生成提示词草案

> A cute frog librarian character in the same warm hand-drawn storybook style, sitting and holding a travel map book. The frog looks focused and curious, as if planning a journey. The map book should have subtle travel markings, folded pages, and a sense of exploration. Gentle green tones, clean readable silhouette, cozy and imaginative feeling, transparent background, suitable for placing in a whimsical website scene.

### 备注

这一张未来可以用在博客页、项目页，甚至做 hover 插图。

## 4）旅行地图册

### 作用

对应项目入口：

- 3D 旅游路线地图
- 杭州巡游计划
- 旅途节点手札

### 目标

做一本非常有“旅行”和“路线标记”特征的地图册。

### 画面内容建议

- 一本地图书，略微打开或半摊开
- 上面有路线红线、小圆点、小旗子、折痕
- 带手绘批注感
- 不要太现代硬核，更像旅行手札 / 地图手册

### 输出建议

- 比例：**4:3**
- 单独物件素材

### 是否透明背景

- **是，透明背景**

### 生成提示词草案

> A whimsical illustrated travel map book, partially open, designed as a project-entry object for a cozy fantasy library. The book should show folded map pages, hand-drawn route lines, little location markers, small flags, and travel notes. It should feel like a journey notebook or route atlas, warm and handcrafted rather than modern UI. Soft paper texture, gentle outlines, slight storybook charm, clean standalone object with transparent background.

### 备注

它应该一眼就能让人联想到“旅行路线 / 地图项目”。

## 5）古旧卷轴 / 世界观地图卷

### 作用

对应项目入口：

- 现实地图魔改小说世界观
- 小说世界观地图志
- 旧城改写计划

### 目标

做一个一看就很有“世界观设定 / 幻想地图 / 古地图改写”感觉的卷轴。

### 画面内容建议

- 羊皮卷半展开
- 有地形线、河流、山脉、手写批注
- 带火漆印或丝带
- 古旧但不要脏乱
- 有“现实地形被幻想化”的气质

### 输出建议

- 比例：**4:3** 或 **3:2**
- 单独物件素材

### 是否透明背景

- **是，透明背景**

### 生成提示词草案

> A magical old map scroll in a warm storybook illustration style. The scroll is partially unrolled, showing fantasy-map details such as mountains, rivers, terrain lines, handwritten annotations, and a sense of worldbuilding. Include subtle wax seal or ribbon details. The object should feel like a project portal for a fictional world setting derived from real geography. Aged parchment texture, elegant and imaginative, transparent background.

### 备注

这个物件最好更偏“幻想设定”，跟旅行地图册拉开气质差异。

## 6）Obsidian 节点笔记本 / 索引册

### 作用

对应项目入口：

- Obsidian 图谱整理
- 知识整理
- 旅行节点抽取实验

### 目标

做一本有“节点、索引、标签、整理方法论”感觉的笔记本。

### 画面内容建议

- 活页夹、标签书、索引册都可以
- 封面有标签纸 / 贴纸
- 打开后能隐约看到分区页
- 页面上可有少量节点连线图样
- 体现“系统整理”的感觉

### 输出建议

- 比例：**4:3**
- 单独物件素材

### 是否透明背景

- **是，透明背景**

### 生成提示词草案

> A hand-drawn indexed notebook or note binder inspired by personal knowledge management and graph-style note systems. The object should include visible tabs, labels, sticky notes, index dividers, and perhaps subtle node-link doodles on paper. It should feel like an organized yet personal Obsidian-style note archive. Warm paper textures, cozy handmade details, readable silhouette, transparent background, suitable as a project-entry object in a whimsical library scene.

### 备注

这是“方法论类项目”的代表物件，要偏“整理工具”而不是普通日记本。

## 7）Three.js 技术机关书

### 作用

- 不要直接变成电子屏幕，而是“魔法技术手册”感

### 输出建议

- 比例：**4:3**

### 是否透明背景

- **是，透明背景**

### 生成提示词草案

> A whimsical technical pop-up book inspired by Three.js experiments and interactive frontend demos. The book is open, and small three-dimensional structures rise from its pages: simple geometric forms, miniature map elements, mechanical pop-up parts, and subtle glowing accents. It should look like a playful hybrid of a technology manual and a magical mechanism book. Keep it warm, imaginative, and visually compatible with a cozy storybook library world. ## 8）松果书屋借阅证 / 挂绳卡片

### 作用

对应你前面讨论的：

- 访客牌
- 借阅证
- 二维码入口
- 联系 / 名片页入口

### 目标

把原本首页不合适的 lanyard card，正式转译为博客页内的“书屋借阅证”。

### 画面内容建议

建议它其实分成 **两面**：

#### 正面

- 青蛙头像 or 青蛙馆长图标
- 标题：Pinecone Library Pass / 松果书屋借阅证
- 小身份信息
- 视觉偏可爱精致

#### 背面

- 中间预留二维码区域
- 少量辅助装饰
- 保持整体风格一致

### 输出建议

- 比例：**2:3**
- 最好做成：
    1. **正面一张**
    2. **背面一张**
- 如果你一定要严格算 8 个素材，可以先只做正面；背面后补

### 是否透明背景

- **是，透明背景**
- 绳子和卡片可以一起出，也可以拆开

### 生成提示词草案

> A whimsical hanging library pass card in a hand-drawn storybook style, designed as a “Pinecone Library Pass” for a cozy frog-run pinecone study. The card hangs from a soft strap or lanyard and feels like a visitor pass or borrowing card. Front side: include a cute frog emblem, title area, and decorative personal-library design elements. Back side: reserve a clean central area for a QR code while keeping the same visual style. Warm, playful, elegant, and suitable as an interactive website object. Transparent background.

### 备注

它未来可以做：

- 正反翻转
- hover 晃动
- 点击显示二维码
- 跳转 guest-card / contact 页面

# 三、建议的生成顺序

我建议不要 8 个一起乱生，而是按这个顺序来：

## 第一轮：先定调

1. 松果书屋主背景
2. 青蛙管理员（躺着读书）
3. 青蛙管理员（抱地图册）

这三张先决定整体世界观和角色气质。

## 第二轮：功能物件

4. 旅行地图册
5. 古旧卷轴
6. Obsidian 节点笔记本
7. Three.js 技术机关书

这四张用来搭 UI 节点。

## 第三轮：特殊交互物件

8. 松果书屋借阅证

最后再做挂绳卡片，因为它需要吃前面已经定好的风格。

# 四、生成时的技术要求建议

为了后续前端好用，最好在提示里尽量强调这些：

## 对单个物件素材

- **transparent background**
- **clean silhouette**
- **centered composition**
- **not a mockup webpage**
- **standalone object**
- **slight shadow allowed**
- **high readability**

## 对背景素材

- **not crowded**
- **leave open spaces**
- **suitable as website background**
- **cohesive lighting**
- **storybook environment**

# 五、最关键的一点：先用“同一张参考图”统一风格

你前面发的那张：

- 青蛙躺在松果屋里看书

那张其实非常适合做**风格锚点**。  
后面如果正式生成这批图，最好尽量让模型围绕这张图的气质延展，而不是每张都重新发散。

也就是说：

- **松果书屋背景**：参考它的温暖剖面感
- **青蛙角色**：参考它的青蛙长相和情绪
- **其他物件**：参考它的线条、配色、童话质感

这样最不容易风格跑偏。
