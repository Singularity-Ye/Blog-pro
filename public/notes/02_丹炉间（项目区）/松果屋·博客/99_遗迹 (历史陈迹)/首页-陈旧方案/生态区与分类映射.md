# 首页 - 生态区与分类映射

> 把小地球仪上的自然区域映射到网站的不同内容分类。

## 映射关系

| 生态区 | 视觉隐喻 | 内容类型 | 路由 |
|--------|----------|----------|------|
| **随笔海洋** | 潮汐、波浪、漂流瓶 | 生活记录、碎碎念、情绪 | `/blog/life` |
| **笔记雨林** | 茂密、生长、缠绕的藤蔓 | Obsidian、知识整理、学习路径 | `/blog/notes` |
| **技术荒漠** | 干燥、风化、绿洲 | 部署、配置、踩坑、修复 | `/blog/tech` |
| **学习雪山** | 高处、冷峻、冰川 | 课程、论文、算法、理论 | `/blog/study` |
| **项目城市** | 建筑、街道、灯火 | 前端、Three.js、创意实验 | `/projects` |

## 设计思路

每个生态区不只是"背景色不同"，而是有独立的：

- 纹理贴图（albedo / noise / mask）
- 地形模型特征
- 颜色体系
- hover 标签样式

### 海洋（已验证）

- 深海 / 浅海 / 海岸泡沫
- 水波纹理、动态流动
- 明亮但不塑料

### 雨林（待开发）

- 绿意充足、有叶片层次
- 避免纯绿色方块感
- 需要 ground / leaf / bark / noise 贴图

### 雪山（待开发）

- 干净冷白、有冰感
- 山体不是灰色锥体
- 需要 snow / ice / rock 贴图

### 荒漠（待开发）

- 温暖、有沙纹、有风吹痕迹
- 不是纯黄方块
- 需要 sand / ripple / rock 贴图

### 城市（待开发）

- 低模建筑、项目入口感
- 有纹理细节但不过于复杂
- 需要 wall / roof / window 贴图
