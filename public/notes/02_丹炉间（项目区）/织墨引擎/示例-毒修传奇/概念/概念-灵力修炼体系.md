# 概念-灵力修炼体系

**苍玄大陆**的修炼者通过吸收天地灵气淬炼己身。正统修炼分为**九境**，每境三重。

## 正统九境

| 境界 | 特征 | 寿元 |
|------|------|------|
| 炼气 | 引灵气入体，淬炼经络 | 120 岁 |
| 筑基 | 凝聚灵基，脱胎换骨 | 200 岁 |
| 金丹 | 结成金丹，灵气自循环 | 500 岁 |
| 元婴 | 丹碎婴成，元神初显 | 1000 岁 |
| 化神 | 元神大成，神念外放 | 2000 岁 |
| 合体 | 身魂合一，法相初成 | 5000 岁 |
| 大乘 | 天地共鸣，法则通明 | 10000 岁 |
| 渡劫 | 天劫淬炼，超脱凡胎 | — |
| 真仙 | 飞升成仙，不入轮回 | 不朽 |

## [[概念-毒修之道]]：正统之外的暗面

毒修并非邪道——它是灵力修炼体系中一条被刻意遗忘的分支。毒修者以**万毒淬体**替代灵气淬炼，每炼化一种奇毒，修为便对应正统境界提升一层。

正统修士视毒修为"捷径"和"邪道"，因为毒修不需要漫长的灵气积累——只需要足够狠的毒，和一颗足够坚韧的心。
