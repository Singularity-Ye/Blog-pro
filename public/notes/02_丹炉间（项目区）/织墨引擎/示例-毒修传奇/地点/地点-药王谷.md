# 地点-药王谷

**药王谷**是南域最核心的炼丹宗门所在地，位于十万大山的腹地，一条狭长的山谷之中。谷口常年被灵雾笼罩，凡人若无引路人，百年难入。

## 谷内布局

```
药王谷
├── 外谷：药市（灵药交易、炼丹材料集散地）
├── 中谷：丹堂（炼丹之所，灵田所在地）
│   ├── 丹阁：藏书千卷，收录历代丹药配方
│   └── 灵田：三百亩灵药种植区
├── 内谷：长老院（谷主与长老居所）
│   └── 禁地：封存禁忌丹方之处
└── 后山悬崖 → 三百里外即为 [[地点-万毒沼泽]]
```

## 禁地传说

药王谷后山禁地封存着一卷被封印的古籍——《万毒真经》。传说三千年曾有弟子翻阅此经后堕入毒道，最终被逐出师门，成为万毒门的创始人。

谷中历代谷主都警告弟子：**"后山禁地，入者逐出。"**

这句话对 [[角色-苏墨]] 而言，不是警告——而是一个预言。
