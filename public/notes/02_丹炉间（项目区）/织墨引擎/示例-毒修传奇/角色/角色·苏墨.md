# 角色·苏墨

> **"如果救人需要千年苦修，而毒能让我三百年就飞升——那我选毒。我没错。"**

## 基本信息

| 属性 | 值 |
|------|----|
| 年龄 | 十九岁 |
| 修为 | 毒藤阶（对应正统金丹期） |
| 体质 | 天生药灵体（万中无一的特殊体质，对药性/毒性有超常感知） |
| 身高 | 清瘦，因常年服毒面色苍白 |
| 特征 | 脖颈处有黑色毒纹蔓延至下颌，平日以高领遮住 |

## 性格

- **坚韧**：从小是孤儿，在药王谷长大——习惯了孤独
- **偏执**：一旦认定方向绝不回头，哪怕全世界反对
- **善良底色**：即使修毒也不曾滥杀——这源于药王谷的教育
- **沉默**：不爱说话，但每一句都掷地有声

## 身世

苏墨是药王谷捡回来的孤儿。养大他的不是哪个人——是药王谷里的药香、丹炉的余温、和那一本被他翻了无数遍的《药王丹经》。

十六岁那年，他在后山采药时坠入禁地，误触了一卷被封印的古籍——《万毒真经》。他发现了一个颠覆认知的事实：**《药王丹经》里最核心的几味药，本身就是剧毒**。

乌头、附子、马钱子、断肠草——药王谷的丹师们小心翼翼地用姜汁炮制、用灵火炙烤，以去除毒性。但如果反过来呢？如果保留毒性，以药引激发——会不会炼出比丹药更强的效果？

他偷尝了一颗自己炼的"毒丹"。

那晚，他的灵脉从未如此通畅。

也就在那晚，他的秘密被师兄发现了。

## 功法

- **基础**：《药王丹经》（药王谷传承，苏墨只学了前半部）
- **核心**：《万毒真经》（万毒门秘传，正逐渐参透）
- **自创**：以淬毒银针为媒介，将毒性精准打入对手灵脉
