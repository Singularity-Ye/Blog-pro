# 节点连通性检测 (Node Connectivity Check)

基于 NetworkX 构建的角色/事件/地点图，检测节点的活跃度与连通性异常。

## 检测指标

| 指标 | 计算方式 | 异常阈值 |
|------|---------|---------|
| **出度 (Out-Degree)** | 节点向外链接的章节数 | 连续 10 章为 0 → 雪藏警告 |
| **入度 (In-Degree)** | 引用该节点的章节数 | 整部作品为 0 → 孤立节点 |
| **度变化率** | 滑动窗口内度数的变化斜率 | 斜率突变 > 50% → 异常标注 |
| **中介中心性** | 节点在最短路径上的出现频率 | 极高 → 核心角色；极低 → 边缘角色 |

## 可视化输出

在前端图谱面板中，角色节点的大小与度成比例——主角最大，龙套最小。被雪藏的角色节点呈现灰色。
