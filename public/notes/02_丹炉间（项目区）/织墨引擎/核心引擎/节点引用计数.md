# 节点引用计数 (Node Reference Counting)

统计每个设定节点（[[角色·]]、[[地点·]]、[[事件·]]）被多少章节文件引用，是连通性检测和变更传播的基础数据。

## 核心指标

| 指标 | 含义 | 用途 |
|------|------|------|
| **RC (Reference Count)** | 总引用次数 | 衡量角色的"戏份" |
| **ARC (Active RC)** | 最近 5 章内的引用次数 | 衡量角色的当前活跃度 |
| **RC Slope** | 引用计数的滑动窗口趋势 | 检测"逐渐退场"或"突然消失" |

## 数据流

```
Obsidian Vault → Wiki-link 正则提取 → 
  {节点ID: [引用文件列表]} → 
    RC 生成器 → 
      引用计数报告 JSON → 
        前端热力图 / 雪藏警告
```
