# 因果链追溯 (Causal Chain Tracing)

从结局事件节点出发，对全图执行反向 BFS，验证每条因果链的完整性。

## 算法

```
输入：终点事件 E_end
输出：因果完整性报告

1. 队列 Q = [E_end]
2. while Q 非空:
3.   E = Q.pop()
4.   for each cause_node ∈ E.causes:
5.     访问 cause_node，标记为"已追溯"
6.     将 cause_node 加入 Q
7. 遍历所有事件节点：
8.   if 节点未被标记 && 节点有 effects 指向已标记节点:
9.     报告："孤立因果链片段"
```

## 常见发现

- **断链**：事件 A → 事件 C，但缺少中间的 B
- **循环因果**：A 导致 B，B 导致 A（时间旅行情节除外）
- **未回收伏笔**：事件标记为 `type: foreshadowing` 但无后果事件链接

## 与 [[时间线冲突检测]] 的协同

因果链验证逻辑正确性，时间线验证时序合理性。一个事件可以有正确的因果但错误的时序。
