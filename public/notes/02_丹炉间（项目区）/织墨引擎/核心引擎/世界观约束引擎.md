# 世界观约束引擎 (World Rules Engine)

管理作品中的**非角色规则**——物理法则、魔法体系、社会规则、历史事实的声明与自动校验。

## 规则声明格式

```yaml
world_rules:
  - id: "magic_cost"
    statement: "使用任何魔法都会消耗同等价值的生命力"
    scope: global
    exceptions: []
  - id: "teleport_ban"
    statement: "雾隐镇方圆百里内无法使用传送魔法"
    scope: "[[地点·雾隐镇]]"
    valid_from: "[[事件·古城封印]]"
```

## 校验逻辑

1. 新内容中检测到魔法描写 → 搜索 `domain: magic` 规则
2. 若设为 `scope: global` → 全局生效
3. 若设为 `scope: 地点节点` → 检查当前场景是否在该地点
4. 若有 `exceptions` → 检查当前上下文是否匹配豁免条件

属于守护者 L4 层级——违反世界观规则将触发**阻断生成**。
