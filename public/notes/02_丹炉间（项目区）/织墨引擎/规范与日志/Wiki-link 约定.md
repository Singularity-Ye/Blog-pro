# Wiki-link 约定 (Wiki-link Convention)

WeaveInk 的双链语法规则——所有节点互链的统一格式。

## 格式规范

| 目的 | 语法 | 示例 |
|------|------|------|
| 链接到角色 | `[[角色·名称]]` | `[[角色·林夜]]` |
| 链接到地点 | `[[地点·名称]]` | `[[地点·雾隐镇]]` |
| 链接到事件 | `[[事件·名称]]` | `[[事件·帝都沦陷]]` |
| 链接到章节 | `[[第N章：标题]]` | `[[第3章：废墟中的重逢]]` |
| 带别名引用 | `[[节点名\|显示文本]]` | `[[角色·林夜\|他]]` |
| 块引用 | `[[节点名#标题]]` | `[[角色·林夜#背景故事]]` |

## 禁止格式

- ❌ `[角色名](app://obsidian.md/...)` — 冗余外链
- ❌ `[角色名](file:///...)` — 绝对路径
- ❌ `[[不存在的节点]]` — 悬空链接（先创建节点再引用）

## 解析正则

```python
WIKI_LINK_PATTERN = re.compile(r'\[\[([^\]|#]+)(?:[|#][^\]]+)?\]\]')
# 提取 "角色·林夜" 部分，忽略别名和锚点
```
