# 概念-风水秘术

> **"寻龙分金看缠山，一重缠是一重关。关门若有千重锁，定有王侯居此间。"** —《十六字阴阳风水秘术》

**风水秘术**是摸金校尉一脉传承千年的核心技艺——以山川形势为线索，推断地下陵墓的位置、深度和结构。它不仅是盗墓的技术基础，更是一套完整的古代地理学与能量学体系。

## 十六字真诀

| 诀 | 含义 | 用途 |
|----|------|------|
| **寻龙** | 寻找山脉走向（龙脉） | 锁定大范围陵区 |
| **点穴** | 在龙脉上确定陵墓精确位置 | 精确定位入口 |
| **分金** | 以罗盘测定方位和深度 | 确定墓道走向 |
| **定穴** | 判断墓室类型和墓葬规格 | 评估风险等级 |
| **望气** | 观察山间云雾、植被异常 | 探测墓中毒气/瘴气 |
| **断水** | 分析地下水脉走向 | 预判墓内积水 |
| **辨土** | 分析土壤颜色、质地、气味 | 判断墓葬年代 |
| **识砂** | 识别封土堆的类型和年代 | 判断墓主身份 |

> 《十六字阴阳风水秘术》完整本已失传。现存于世者多为残卷。[[角色-顾北川]] 手中所持即为其祖父留下的残本——缺了最后四字。

## 与 [[概念-阴阳倒转]] 的关系

风水秘术解释"正常"的陵墓。但当墓中出现**阴阳倒转**现象时——所有的风水规则都会被逆转。龙脉变死脉，生门变死门。

古人建造某些大墓时，会故意利用阴阳倒转来保护核心墓室：**你按风水规律找到的位置——恰好是陷阱。**

这就是为什么摸金校尉的祖训是："遇阴阳逆行之墓，弃之勿取。"
