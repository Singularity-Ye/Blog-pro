# 地点-骊山疑冢

**骊山疑冢**位于 [[地点-秦岭]] 骊山北麓一处无名山谷中。三年前被地质勘探队的声呐设备意外发现——它的深度、规模、布局——完全不符合任何已知朝代的墓葬制度。

## 异常之处

| 异常项 | 细节 | 推测 |
|--------|------|------|
| 深度 | 距地表 200 米（正常帝王陵不会超过 50 米）| 在刻意躲避什么 |
| 墓道方向 | 朝向西北（正常帝陵朝南或朝东）| 或为镇物布局 |
| 殉葬坑数量 | 声呐探测到至少十二个独立殉葬坑 | 远超任何已知帝王陵 |
| 金属探测 | 整座墓室被一层金属包裹——疑似铅 | 防辐射？还是封什么在里面？ |

## 内部结构（根据初步勘探）

```
骊山疑冢
├── 外层：十二殉葬坑（环形围绕主墓室）
│   ├── 坑1-4：人骨排列成阵（疑为某种封印仪式）
│   ├── 坑5-8：武器坑（刀剑堆积，几乎所有兵刃都断裂）
│   └── 坑9-12：尚未勘探
├── 中层：铅壁（包裹整个主墓室，厚度超过一米）
└── 核心：主墓室——还没进去过
```

## 老魏的判断

[[角色-老魏]] 看过勘探报告后的第一句话是："这他妈不是墓。这是个锁。"

他指着殉葬坑的布局图："你看——十二个殉葬坑不是随便挖的。它们在按天干地支排列。这十二个坑不是给墓主殉葬的——它们是用来形成'十二地支封印'的。"

"封什么？"

老魏沉默了很久。

"封一个——用秦始皇的名头都压不住的东西。"
