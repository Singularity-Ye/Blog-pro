# 事件·第一夜

老赵在天黑之前下山了。他说他在山腰的石头小屋过夜——明早再上来。马总骂了他两句，没留。林深看到他下山时的步伐——不对。不是走。是逃。像有只眼睛盯着他的后背。

马总睡在 A 栋二楼的医生宿舍。林深睡在隔壁。

凌晨 1:17。林深听到了咳嗽声。不是人的——是一个小孩。小女孩。从 B 栋的方向传来。他在黑暗中睁开眼。他的右耳听得格外清楚——那个方向，那个频率。十二声咳嗽，每一声间隔完全一致。十二声后——停了。过了大约两分钟。咳嗽声再次开始。同样的声音，同样的频率。是录音。是回放。是某种东西在模仿——但它不知道什么叫"随机"。所以它重复得一模一样。

他起身想去叫马总——马总的房门开着。人不见了。林深在 B 栋一楼走廊尽头找到了他——蹲在墙角。公文包扔在脚下。不眨眼。不说话。

凌晨 2:44。金属器械碰撞声从 C 栋传来。声音穿过铅封的门缝——冷冷的、没有回音——像是从另一个空间渗透过来的。林深的右耳捕捉到了不一样的东西。不是哭声。是说话声。一个字。反复。重复。

凌晨 4:04。老赵从山下跑了回来——跑着上来的。他用登山绳从窗户爬进来，喘得像个漏斗。他说他听到林深的喊声了——林深没有喊过。但老赵坚持说听到了。三个人在楼梯间交汇。林深后来回忆——那是那晚唯一令他真正恐惧的瞬间。不是看到什么——是意识到自己和老赵听到的是同一种声音。但那种声音——不可能被听到。
