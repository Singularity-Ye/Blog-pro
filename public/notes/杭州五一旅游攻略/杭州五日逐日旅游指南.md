# 杭州五日逐日旅游指南

> ⚡ 核心法则：地下 > 水上 > 步行 > 两轮 > 四轮。每天 6:30 前出门避开人流。

---

## Day 1：西湖经典环线（5月1日）

**主题**：断桥 → 白堤 → 苏堤 → 南山路 → 老城夜市
**强度**：步行约 12km，中等

### 行程

| 时间 | 地点 | 笔记 |
|------|------|------|
| 6:00 | 起床 | 🔴 必须早起 |
| 6:30 | **[[断桥残雪]]** | 只有这个时间点才没人。拍照 20min |
| 7:00 | 白堤 → 孤山 | 步行约 40min。顺路看 [[平湖秋月]] |
| 8:00 | [[岳王庙]] | ¥25，30min |
| 8:30 | [[曲院风荷]] → [[苏堤]] | 从北口进苏堤，全程 2.8km 步行约 1h |
| 10:00 | 花港观鱼 | 免费，30min |
| 10:30 | [[净慈寺]] | ¥10，"南屏晚钟"。出口买 [[净慈寺素烧鹅]] ¥10 |
| 11:30 | 🍜 **午餐** | 选项A：净慈寺素烧鹅（省时间） / 选项B：[[新丰小吃]]（湖滨分店，¥20/人） |
| 13:00 | [[雷峰塔]] | ¥40，登塔看西湖全景 |
| 14:30 | [[太子湾公园]] | 免费，人少 |
| 16:00 | [[苏堤]]南口 → [[河坊街]] | 步行 5min |
| 17:00 | 🍜 **晚餐** | [[福缘居]]（脆皮大肠 ¥68，⚠️ 16:30 去排第一波） |
| 19:00 | [[南宋御街]] | 消食散步 |
| 20:00 | 🍺 夜宵 | [[吴山广场]]/河坊街夜市（只逛不吃主街的游客货） |

### 🟢 备用方案（如果苏堤人多）

拐进杨公堤 → [[浴鹄湾]]（免费秘境）→ [[郭庄]]（¥10，喝茶看湖）

### 🍜 今日美食锚点

- 早餐：提前备好 [[知味观]] 定胜糕
- 午餐：[[净慈寺素烧鹅]] 或 [[新丰小吃]]
- 晚餐：[[福缘居]]（早去！）

## Day 2：灵隐佛国 + 龙井徒步（5月2日）

**主题**：灵隐寺 → 法喜寺 → 龙井村 → 九溪十八涧
**强度**：步行约 12km，中等偏上（含大量台阶和山路）

### 行程

| 时间 | 地点 | 笔记 |
|------|------|------|
| 5:30 | 起床 | 🔴 比 Day 1 更早！ |
| 6:00 | 🚗 打车出发 | 目的地：飞来峰景区大门。**不要坐 7 路公交！** |
| 6:30 | [[飞来峰]] 排队 | 7:00 开门，6:30 到才能第一批进 |
| 7:00 | 飞来峰石窟 | 🆕 **免费**（需预约），游览 1h |
| 8:00 | [[灵隐寺]] | +¥30 香花券。进门先领三支清香 |
| 9:30 | 🍜 [[灵隐寺十方苑]] | 罗汉面 ~¥30。趁早吃避免排队 |
| 10:30 | 灵隐寺深度游览 | 大雄宝殿 → 药师殿 → 华严殿 |
| 11:30 | → [[法喜寺]] | 步行天竺路 20min 或观光车 ¥10 |
| 12:00 | 🍜 [[法喜寺]] 斋堂 | **¥5 素斋**，管饱。网红机位飞檐长廊拍照 |
| 13:30 | → [[龙井村]] | 打车 15min 或步行 40min |
| 14:00 | 🍵 龙井村 | 找正规茶馆喝杯正宗龙井（¥30-50/杯） |
| 15:00 | → [[九溪十八涧]] | 开始徒步，一路下坡约 1.5-2h |
| 17:00 | 九溪烟树 | 瀑布拍照 |
| 17:30 | 九溪公交站 | 🔴 从这里出去，不要打车进九溪路！ |
| 18:30 | 🍜 **晚餐** | 回到市区，[[菊英面馆]] 片儿川 ¥25 |

### 🔴 今日铁律

- 地铁不到灵隐！打车直接去飞来峰大门
- 九溪路极窄，从九溪公交站徒步进入
- 备足现金（香花券可能不收移动支付）
- 🥾 穿防滑徒步鞋（十里琅珰石板路青苔多）
- 🦟 带驱蚊液
- 💧 带 2 瓶水（十里琅珰山顶无商店）

### 🟢 体力不足时的精简版

```
飞来峰 → 灵隐寺 → 法喜寺（斋饭）→ 法喜寺拍照 → 回市区
（砍掉龙井村和九溪，节省 3h+）
```

### 🍜 今日美食锚点

- 早餐：提前备干粮
- 午餐：[[灵隐寺十方苑]] 罗汉面 或 [[法喜寺]] ¥5 素斋
- 晚餐：[[菊英面馆]] / [[方老大面]] / [[奎元馆]]

## ⚡ 每日铁律（再说一遍）

1. **6:30 前到断桥/灵隐**，8 点已是人海
2. **能地铁绝不上路面**，7 路公交是五一死锁陷阱
3. **龙翔桥站=别下车**，改凤起路/定安路
4. **灵隐寺免费需预约**：2025.12起免票！「杭州灵隐飞来峰」小程序提前预约，五一限额 6.5 万
5. **苏堤/白堤禁行单车**，穿最舒服的鞋
6. **预约提前 5 天+**，当天约=已售罄
7. **水上巴士 3 元**是最被低估的交通方式
