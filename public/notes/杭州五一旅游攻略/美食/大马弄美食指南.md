# 大马弄美食指南

杭州老底子居民的菜市场，藏在 [[河坊街]] 旁仅 3 分钟步行距离——吃到的却是两个世界的东西。

## 为什么不去河坊街主街

| | 河坊街主街 | 大马弄 / 中山南路 |
|------|-----------|-----------------|
| 定位 | 游客街 | 本地居民日常 |
| 价格 | 20-30 元/份 | 5-15 元/份 |
| 内容 | 老北京冰糖葫芦、轰炸大鱿鱼 | 葱包烩、油墩儿、定胜糕 |
| 正宗度 | ❌ 全国景区通用款 | ✅ 杭州老底子味道 |

## 推荐小吃

| 食物 | 说明 | 参考价 |
|------|------|--------|
| **葱包烩** | 杭州传统小吃，薄饼裹油条+葱，压烤至酥脆 | 5-8 元 |
| **油墩儿** | 萝卜丝炸饼，外酥里嫩 | 3-5 元 |
| **定胜糕** | 米粉蒸糕，豆沙馅，粉红色 | 5-10 元 |
| **片儿川** | 杭州特色面，雪菜+笋片+肉丝 | 15-20 元 |
| **虾爆鳝面** | 老杭州硬菜面 | 30-40 元 |

## 怎么走

```
河坊街主街 →[步行 3min]→ 大马弄（菜市场方向）
                ↓
           中山南路（面馆集中地）
```

## 注意事项

- 大马弄是菜市场，环境不如河坊街精致——但这恰恰是它正宗的原因
- 大多数摊位只收现金/微信，准备零钱
- 早上 7:00-10:00 最热闹，中午后很多摊位收摊

## 🍜 杭州面馆指南

片儿川和拌川是杭州面食的灵魂。找面馆有一个黄金法则：**不用迷信必吃榜。**

| 面 | 浇头 | 参考价 | 口味 |
|----|------|--------|------|
| **片儿川** | 雪菜 + 笋片 + 肉丝 | 15-20 元 | 咸鲜，不吃笋/雪菜的慎点 |
| **虾爆鳝面** | 虾仁 + 鳝段 | 30-40 元 | 老杭州硬菜面 |
| **拌川** | 炒浇头干拌 | 15-25 元 | 浓郁入味 |

### 怎么找正宗面馆

去居民区楼下（**朝晖小区、复兴路**附近），看到招牌"XX面馆"且里面坐满了大爷大妈——进去吃绝对不会错。

## 🛍️ 伴手礼去哪买

> 千万别在河坊街或雷峰塔底下买几十一盒的"特产糕点"。

| 渠道 | 推荐 |
|------|------|
| 🥇 **世纪联华超市** | 杭州本地最大连锁，西湖藕粉、定胜糕全有，本地人价格（推荐庆春广场店/西湖文化广场店） |
| 🥈 **知味观** | 老字号点心铺 |
| 🥉 **农贸市场** | 散装茶叶、藕粉 |

### 认准品牌

| 特产 | 正确品牌 | 避坑 |
|------|---------|------|
| 西湖藕粉 | **三家村** 或 **天堂牌** | ❌ 景区杂牌 = 智商税 |
| 定胜糕/绿豆糕 | **知味观** | ❌ 河坊街散装糕点过甜且贵 |
| 龙井茶 | 茶农家或有资质茶庄 | ❌ 路边"免费品茶" = 购物陷阱 |
