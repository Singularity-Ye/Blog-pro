# 菊英面馆

三十多年网红老店。汤头用火腿骨慢熬，鲜掉眉毛。杭州片儿川的标杆之一——很多人来杭州吃的"第一碗片儿川"就在这里。

## 推荐

| 菜品 | 价格 | 笔记 |
|------|------|------|
| **片儿川** | ~¥25 | 火腿骨汤头，笋片+雪菜+肉丝，极鲜 |
| 拌川 | ~¥25 | 炒浇头干拌，锅气足 |
| 大排面 | ~¥30 | 大排扎实，外酥里嫩 |

## 基本信息

- **位置**：中山北路（近鼓楼/中山南路片区）
- **人均**：¥25-35
- **营业时间**：约 6:30 - 13:00，17:00 - 20:00（午休时段不营业）
- **交通**：地铁 5 号线江城路站步行 8min

## 🍜 片儿川详解

片儿川是杭州独有的面——**全国其他城市基本吃不到正宗的**。

| 要素 | 说明 |
|------|------|
| 面条 | 碱水细面，口感爽滑 |
| 汤底 | 火腿骨+猪骨慢熬 4h+，清澈鲜甜 |
| 浇头 | 雪菜+笋片+瘦肉丝（经典三件套） |
| 笋 | 春笋/鞭笋最佳，冬笋次之 |

> 💡 **五一时节**：春笋季尾巴，正是吃片儿川的最佳时段之一。

## 🟢 串联建议

```
方案 1（老城早餐线）：
游埠豆浆（7:00 咸豆浆+油条）→ 步行 5min → 菊英面馆（8:30 片儿川当早午餐）
    → 步行 3min → 孙奶奶葱包烩 → 大马弄 → 河坊街

方案 2（面馆巡礼）：
菊英面馆（片儿川）→ 步行 10min → 方老大面（虾爆鳝，如果排队能接受）
    → 步行 15min → 奎元馆（虾爆鳝面）
```

## ⚠️ 五一提醒

- 五一人多，但没有 [[方老大面]] 那么夸张
- 🟢 建议 **6:30 开门就去**（杭州人吃面很早）
- 如果排队 30min+ → 对面 [[孙奶奶葱包烩]] 先买一份垫垫
- 菊英面馆是全天营业（午休 13:00-17:00），错过早市午市也能吃到
- 片儿川配一份拌川是最佳组合——一碗汤一碗干，两种体验
