# 夜游西湖

五一期间西湖夜间特别活动，灯光+音乐+夜景，是杭州最浪漫的夜间体验之一。

## 核心体验

### 🛶 夜游西湖游船

- **路线**：湖滨 → 三潭印月 → 断桥 → 白堤 → 返回
- **时间**：18:30-21:00（末班）
- **票价**：¥50-80/人（根据船型）
- **码头**：湖滨一公园 / 中山公园 / 杭饭码头

### 🌉 夜间步行路线

```
断桥残雪 → 白堤 → 孤山 → 西泠桥 → 北山街 → 宝石山看夜景
    ↓                                              ↓
  平湖秋月（拍照）                            保俶塔亮灯
```

### 🎵 湖滨音乐喷泉

- **位置**：湖滨三公园
- **时间**：每晚 19:00 / 20:00（各一场，约 15 分钟）
- **门票**：免费
- **⚠️ 五一提醒**：人山人海，提前 30 分钟占位

## 夜游建议

- ⏰ 最佳时间：18:30-21:00
- 📷 带三脚架/手机稳定器（夜景拍摄）
- 🧥 夜晚湖边风大，带薄外套（参见 [[杭州5月穿搭指南]]）
- 🍜 夜游后可去 [[武林夜市]] 或 胜利河美食街 吃宵夜
- 💡 北山街/宝石山是看西湖夜景全貌的最佳位置

## 适合人群

- 💑 情侣约会首选
- 📷 夜景摄影爱好者
- 🧘 喜欢安静散步的人（避开音乐喷泉区域即可）

## ⚠️ 五一提醒

- 夜游人非常多，尤其是音乐喷泉和断桥
- 建议避开 19:00-20:30 高峰，选择 20:30 后漫步
- 打车困难，提前约车或步行至地铁站（凤起路站/龙翔桥站）
