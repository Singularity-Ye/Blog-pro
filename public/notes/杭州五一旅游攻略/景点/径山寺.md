# 径山寺

径山万寿禅寺，日本茶道和临济宗的祖庭。位于径山山顶，远离市区喧嚣，是杭州寺庙中最清幽的一座。

## ⚠️ 票价纠错

> 原图标注"径山：免费"——这是**不准确的**。

- **径山风景区**：免费进入
- **径山寺（核心寺庙）**：门票 **30 元**，需另行购买

## 基本信息

- **位置**：余杭区径山镇径山顶
- **交通**：离市区约 1 小时车程，建议自驾或跟团
  - 自驾：市区→杭长高速→径山出口→盘山路至山顶
  - 公交：地铁 2 号线至良渚站→换乘公交至径山（耗时 2h+）
- **开放时间**：7:30 - 17:00
- **游览时间**：3-4h（含交通）
- **上山**：需换乘景区接驳车（另收 ¥10-15）

## 核心看点

### 🍵 茶文化圣地

| 看点 | 说明 |
|------|------|
| 径山茶 | 寺庙自产，可在寺内品尝（¥30-50/杯） |
| 茶宴体验 | 寺庙素斋+径山茶组合（需预约） |
| 日本茶道起源 | 南宋时日本僧人在此学茶，带回日本成为茶道 |
| 陆羽泉 | 茶圣陆羽著《茶经》之处，就在径山附近 |

### 🏛️ 寺庙建筑

| 区域 | 说明 |
|------|------|
| 大雄宝殿 | 主殿，建筑宏伟 |
| 钟楼 | 可登楼远眺 |
| 径山古道 | 上山步道，两侧竹林茂密（约 40min） |
| 凌霄阁 | 最高点，俯瞰余杭全景 |

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 径山古道 | 竹林+石板路+远山 | 竖构图，纵深感 |
| 寺庙屋顶 | 层层叠叠的飞檐 | 广角，俯拍 |
| 茶室窗边 | 茶杯+窗外竹林 | 框架构图 |
| 凌霄阁远眺 | 余杭平原全景 | 天气好时拍 |

## 🟢 推荐路线

```
停车场 → 接驳车上山（15min）→ 寺庙入口
    → 大雄宝殿（20min）→ 钟楼（10min）
    → 茶室喝径山茶（30min）← 重点！
    → 凌霄阁远眺（15min）→ 径山古道下山（40min，或坐接驳车）
```

## ⚠️ 五一提醒

- 径山寺五一人会比平时多，但**比灵隐寺好太多**
- 🟢 如果灵隐寺约不到 → 径山寺是最好的替代方案
- 上山盘山路弯多，晕车的人提前吃晕车药
- 山顶温度比市区低 3-5°C，带件外套
- 寺庙素斋需提前预约，五一可能满
- 💡 径山适合"不想挤人"的人——如果你想去一个清净的寺庙，这里是最佳选择
