# 南宋德寿宫遗址博物馆

南宋高宗、孝宗退位后的居所遗址。2022 年开放后迅速成为杭州最难预约的博物馆之一。数字光影还原宋代宫殿——"德寿夜宴"是夜游王牌。

## 基本信息

- **门票**：免费，但必须在公众号**提前预约**
- **位置**：上城区望江路 228 号
- **交通**：地铁 5 号线「江城路」站步行 5min
- **开放时间**：9:30 - 17:00（周一闭馆，五一正常开放）
- **游览时间**：1.5-2h

## 🔴 五一预约

五一期间预约名额在放票当天**秒空**。建议提前 **5-7 天**在"南宋德寿宫遗址博物馆"公众号抢票。

| 策略 | 说明 |
|------|------|
| 提前预约 | 五一需提前 5-7 天 |
| 放票时间 | 每天 8:00 放新号 |
| 约不到？ | 改去 [[杭州博物馆]]/中国丝绸博物馆/浙江省博物馆 |
| 候补 | 偶尔有人退票，可以刷一刷 |

## 核心看点

### 🏛️ 遗址展示

| 看点 | 说明 |
|------|------|
| 重华殿遗址 | 南宋宫殿基址，地面玻璃下可见 |
| 慈福宫遗址 | 太后居所 |
| 后苑遗址 | 御花园遗迹 |
| 数字复原 | 声光电还原宋代宫殿原貌 |

### 🌙 德寿夜宴（夜场王牌）

| 项目 | 说明 |
|------|------|
| 时间 | 夜场，需单独预约 |
| 内容 | 数字光影+沉浸演艺+NPC 互动 |
| 特色 | 穿越回南宋宫廷的感觉 |
| 预约 | 比日场更难约——秒空 |

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 遗址展厅 | 玻璃下的南宋基址 | 俯拍，展示规模 |
| 数字复原区 | 声光电投影 | 抓拍光影瞬间 |
| 建筑外观 | 现代+古典融合设计 | 正面拍，简洁 |
| 庭院 | 枯山水+宋式庭院 | 极简构图 |

> 💡 **穿搭建议**：宋制汉服/素色衣服——在宋韵场景中特别出片。

## 五一特展

- **宋代官式建筑展**：德寿宫×宁波保国寺跨城对话
- **"万宠雅集"国风市集**：宋韵宠物文创
- **"德寿夜宴"**：数字光影+沉浸演艺+NPC 互动（夜场需单独预约）
