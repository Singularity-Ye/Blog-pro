# 太子湾公园

西湖边的免费公园，以春季郁金香和樱花闻名。五一期间花期已过，但公园本身的草坪、溪流和园林景观依然值得一逛——且人比苏堤少得多。适合苏堤走完后"躺平"休息。

## 基本信息

- **门票**：免费
- **位置**：南山路，[[雷峰塔]] 以南
- **最佳季节**：3 月下旬-4 月中旬（郁金香+樱花花期）
- **游览时间**：30-60min
- **交通**：公交 4 路/12 路至太子湾站 / 苏堤南口步行 15min

## 核心看点

| 看点 | 说明 | 五一状态 |
|------|------|:--:|
| 🌷 郁金香花海 | 杭州最大郁金香展 | ❌ 已谢 |
| 🌸 樱花 | 大片樱花林 | ❌ 已谢 |
| 🌳 大草坪 | 超大草坪，可躺平 | ✅ 绿草如茵 |
| 💧 叠瀑溪流 | 人造瀑布+溪流 | ✅ |
| 🏠 荷兰风车 | 公园标志建筑 | ✅ |
| ⛪ 教堂式建筑 | 婚纱取景热门 | ✅ |

> 💡 五一虽然花没了，但**大草坪+溪流+风车**的组合依然很美——而且人少（大家都去看花了）。

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 风车+大草坪 | 公园最出片的组合 | 广角，人物+风车+蓝天 |
| 叠瀑溪流 | 水流+石头+绿树 | 低角度拍水流 |
| 教堂式建筑 | 欧式建筑+草坪 | 正面拍，经常有婚纱 |
| 草坪上躺平 | 天空+树冠 | 仰拍，极简构图 |
| 溪流石桥 | 小桥+流水+树荫 | 侧拍，日系感 |

> 💡 **穿搭建议**：日系/田园风——大草坪+风车背景太搭了。白色连衣裙最百搭。

```
    → 步行 10min → 郭庄
```

## ⚠️ 五一提醒

- 郁金香花期已过，但草坪和溪流景观依然不错
- 人流比苏堤/白堤**少 50% 以上**——是五一西湖边少有的"能躺平"的地方
- 🟢 可从太子湾转入 [[浴鹄湾]]（杨公堤方向）或 [[九溪十八涧]]
- 适合作为苏堤徒步后的"中场休息"——躺在草坪上歇脚
- 周末草坪上可能有家庭野餐——工作日更安静
- 园内没有太多遮阳——五月太阳已经很毒，带帽子
