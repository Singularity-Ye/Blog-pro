# 雷峰塔

西湖十景"雷峰夕照"所在地。2002 年在原址重建，钢结构+铜饰，登塔可 360° 俯瞰西湖全景。塔内展示原雷峰塔地宫出土的珍贵文物。

## 基本信息

- **门票**：¥40
- **位置**：南山路 15 号（[[净慈寺]] 马路对面）
- **开放时间**：8:00 - 17:30（🆕 五一延长至 **20:00**，19:30 停止售票）
- **登塔**：有电梯！不用爬楼梯

## 🕐 最佳时间

- 🌇 **黄昏前 1 小时（约 16:30）**：登塔看"雷峰夕照"——西湖十景中色彩最浓烈的一幕
- 上午也可，但逆光拍西湖效果差

## 南山路集群

```
苏堤南口 →[步行 5min]→ 净慈寺 →[步行 5min]→ 雷峰塔 →[步行 8min]→ 太子湾公园
```

## 📸 拍照点

| 机位 | 说明 |
|------|------|
| 塔顶（5 层） | 360° 西湖全景，苏堤+三潭印月+湖心亭尽收眼底 |
| 塔前广场 | 仰拍雷峰塔全景 |
| [[净慈寺]] 方向 | 从净慈寺门口拍雷峰塔+南屏晚钟同框 |
| 长桥公园 | 经典机位，拍雷峰塔倒影+夕阳 |

## 🍜 隐藏美食

> 🔥 雷峰塔对面 [[净慈寺]] 出口处有 **[[净慈寺素烧鹅]]** ¥10/份——买一盒坐在南山路长椅上看雷峰塔吃，比花几百块去网红店香一万倍。

## 塔内看点

| 楼层 | 看点 |
|------|------|
| 底层 | 原雷峰塔遗址（玻璃地面，能看到倒塌的塔基） |
| 2 层 | 东阳木雕《白蛇传》全景 |
| 3-4 层 | 西湖全景（每层角度不同） |
| 5 层 | 最高观景台，金色塔顶 |

## 注意事项

- 五一排队登塔可能需要 30-60 分钟
- 黄昏时段人最多（都来看雷峰夕照）
- 🟢 如果排队太长：在塔前广场拍外观即可，登塔视野虽好但非必须
- 塔内电梯排队可能比楼梯还长
