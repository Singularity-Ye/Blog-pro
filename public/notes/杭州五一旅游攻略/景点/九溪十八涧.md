# 九溪十八涧

杭州最经典的徒步路线之一。溪水穿路而过，两旁茶园和古树交错。"九溪烟树"是西湖新十景之一。全程免费，适合所有年龄段——夏天还能踩水玩。

## 基本信息

- **门票**：免费
- **位置**：西湖区九溪路
- **最佳季节**：四季皆宜，五一水量适中，夏日可踩水
- **游览时间**：1.5-2.5h（单程 5km）
- **交通**：公交 4 路/308 路至九溪站

## 🟢 五一交通提示

> 九溪路**极其狭窄**（基本是单车道），五一期间必定堵死。

- ✅ **正确方式**：从**九溪公交站**下车，徒步或扫共享单车进入
- ❌ **错误方式**：打车/网约车直接定位"九溪烟树"——车会堵在路上动弹不得

## 路线

### 方案 1：从龙井村进入（推荐 🟢）

```
龙井村 → 九溪十八涧 → 九溪烟树 → 九溪公交站
全程约 5km，步行 1.5-2h，一路下坡
```

### 方案 2：从九溪公交站进入

```
九溪公交站 → 九溪烟树 → 九溪十八涧 → 龙井村
全程约 5km，一路缓坡上，约 2-2.5h
```

> 🟢 推荐从龙井村下坡走，节省体力。

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 九溪烟树八觉亭旁 | 瀑布+碧绿深潭（"小九寨沟"）| 低角度拍瀑布 |
| 九溪十八涧踩石过溪 | 古树+溪流+石头 | 经典构图，蹲下拍水面 |
| 网红大树根 | 巨大树根+溪流 | 广角，展示规模 |
| 一号桥 | 石桥+溪流+茶田 | 侧拍，纵深感 |
| 龙井村观景亭 | 茶田+远山 | 远景，层叠感 |

> 💡 **穿搭建议**：防滑鞋必备！石板路+溪水=容易滑倒。穿速干面料，踩水不心疼。

## 🍜 周边美食

- [[龙井村]] 内农家乐和茶馆（注意避坑"免费品茶"）
- 建议自带干粮和水
- 下山后回市区吃 [[菊英面馆]] 或 [[方老大面]]

## ⚠️ 五一提醒

- 九溪路五一大堵车——**公交到九溪站，步行进入**
- 🟢 从龙井村走下来（下坡）比从九溪走上山（上坡）轻松很多
- 石板路青苔多——穿防滑鞋，雨后特别滑
- 蚊虫极多——带强效驱蚊液
- 茶山无遮阳——带帽子+涂防晒
- 如果不想走全程 → 从九溪公交站走到九溪烟树（30min）再原路返回也很美
- 💡 最佳时间：8:00-10:00（人少+光线好+凉快）
