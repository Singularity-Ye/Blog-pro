# 湘湖

西湖的"姊妹湖"，位于萧山区。水域面积堪比西湖，但游客量不到十分之一。五一期间——去西湖是看人，来湘湖才是看湖。

## 基本信息

- **门票**：免费
- **位置**：萧山区
- **交通**：🚇 地铁 1 号线「湘湖」站（从 [[CP32漫展]] 大会展中心站仅 **3 站**）
- **游览时间**：半天-1天
- **推荐游览方式**：骑行环湖（有专用骑行绿道，约 15km，1.5h）

## 🟢 与 CP32 完美串联

```
CP32漫展（大会展中心，5/1-2 或 5/4-5）
        ↓ 地铁 1 号线 3 站
    湘湖站
        ↓
    湘湖（骑行/散步/野餐/看日落）
```

> 💡 漫展散场后别挤地铁回市区——来湘湖骑行看日落。

## 🚴 骑行攻略

| 项目 | 说明 |
|------|------|
| 路线 | 环湖绿道，约 15km |
| 耗时 | 1-1.5h（休闲骑） |
| 租车 | 共享单车随处可见 / 景区租车点（¥20-30/h）|
| 难度 | 平坦，适合所有人 |
| 亮点 | 沿湖骑行，随时停车拍照 |

## 西湖 vs 湘湖

| | 西湖 | 湘湖 |
|------|------|------|
| 水域面积 | 6.4km² | 约等于西湖 |
| 五一拥挤度 | 🔴 挤爆 | 🟢 舒适 |
| 骑行 | ❌ 苏堤/白堤禁行 | ✅ 专用骑行绿道 |
| 古迹密度 | 极高 | 较少 |
| 门票 | 免费 | 免费 |
| 适合 | 打卡经典 | **放松/骑行/野餐** |
| 拍照 | 人文+湖景 | 自然+湖景 |
| 花费时间 | 至少 1 天 | 半天-1天 |

## ⚠️ 五一提醒

- 湘湖五一人不多——游客都去西湖了
- 🟢 骑行绿道五一人也不多，非常适合
- 带野餐垫+零食，在湖边草地上度过一个下午
- 跨湖桥遗址博物馆免费但需预约
- 日落时间约 18:30-19:00——提前找个好位置
- 如果不想回市区吃饭 → 湘湖周边有餐厅，人均 ¥50-80
