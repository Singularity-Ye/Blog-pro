# 香积寺

京杭大运河畔的千年古寺，国内**唯一供奉食神（监斋菩萨）**的寺庙。始建于北宋，2009 年重建后融合了唐宋风格与现代设计——铜质外立面在阳光下金光闪闪，被称为"最美寺庙"之一。

## 基本信息

- **门票**：¥20（一口价，无需预约）
- **位置**：拱墅区丽水路（京杭大运河畔）
- **交通**：地铁 3 号线"香积寺"站 B 出口步行 5min / 水上巴士至信义坊码头步行 10min
- **开放时间**：7:00 - 17:00
- **游览时间**：30-60min

## 核心看点

### 🍳 供奉食神

- 香积寺供奉的是**监斋菩萨**（中国佛教中的"灶神"/食神）
- 来这里祈福的主要是厨师、餐饮从业者
- 可以求"食运"——吃得健康、吃得开心

### 🏛️ 建筑特色

| 建筑 | 特点 |
|------|------|
| 大雄宝殿 | 铜质外立面，阳光下金光闪闪 |
| 钟楼/鼓楼 | 唐宋风格，飞檐翘角 |
| 天王殿 | 供奉弥勒佛+四大天王 |
| 后院花园 | 小桥流水，安静禅意 |

### 🍜 香积寺素面

- **价格**：¥15-20/碗
- **口味**：菌菇汤底，清淡鲜美
- **位置**：寺内斋堂
- ⚠️ 卖完即止，建议上午去
