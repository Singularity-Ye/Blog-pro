# 杭州博物馆

杭州市级博物馆，位于上城区粮道山（吴山），紧邻 [[河坊街]]。馆藏以杭州历史为主线，从良渚文明到近现代。免费参观，是了解杭州的好去处。

## 🔴 拓扑纠错

> **原图标注"西湖 → 杭州博物馆 打车 28 分钟"——全图最离谱的错误之一。**

杭州博物馆就在河坊街旁边的粮道山上。从河坊街步行仅需 **5 分钟**。作者很可能混淆了「杭州博物馆」（市中心粮道山）与「浙江省博物馆之江馆」（之江文化中心，确实较远）。

## 基本信息

- **门票**：免费，需提前在微信公众号预约
- **位置**：上城区粮道山 18 号（⚠️ 不是西湖区）
- **开放时间**：9:00 - 16:30（周一闭馆，五一正常开放）
- **游览时间**：1.5-2.5h
- **交通**：地铁 7 号线吴山广场站步行 5min

| 展览 | 说明 | 推荐 |
|------|------|:--:|
| 杭州历史数字展 | 声光电沉浸体验 | ⭐⭐⭐⭐ |

## 🟢 串联建议

```
方案 1（老城文化半日）：

    → 步行 15min → 西湖（南山路段）

方案 3（亲子线）：
杭州博物馆（9:00，看水晶杯）→ 河坊街（午餐）

```

## ⚠️ 五一提醒

- 🔴 **五一期间极难预约**。建议提前 3-7 天预约
- 现场基本没有余票——不要抱侥幸心理
- 如果没约上，周边还有：
  - [[南宋德寿宫遗址博物馆]]（同样需要预约）
  - 浙江自然博物院（预约压力略小）
  - 中国丝绸博物馆（预约压力略小）
- 周一闭馆（五一期间 5/5 是周一，注意！）
- 馆内空调很足——夏天进去是避暑好去处
