# 梅家坞

西湖龙井茶的核心产区之一，与 [[龙井村]] 齐名但更安静、更原生态。沿着梅灵路一路骑行或步行，两侧茶山连绵，是杭州人周末喝茶发呆的私藏地点。

## 基本信息

- **位置**：西湖区梅灵路（龙井村与灵隐寺之间）
- **门票**：免费
- **交通**：公交 103 路/121 路到梅家坞站，或从[[龙井村]]骑行 20min
- **推荐时长**：1.5-3h
- **最佳季节**：3-4 月明前茶季 / 10-11 月秋茶季

## 核心体验

### 🍵 喝茶

| 项目 | 说明 |
|------|------|
| 价格 | ¥30-80/杯（明前茶更贵，¥100+/杯）|
| 形式 | 农家茶室，坐在院子里喝茶看茶山 |
| 防坑 | **先问价再坐下**，不要被"免费品尝"套路 |
| 推荐 | 找一家有院子的茶室，看茶山发呆 |

> 💡 **最佳体验**：下午 14:00-16:00，找一家有院子的农家茶室，点一杯龙井，什么都不做——这就是杭州人说的"喫茶去"。

### 🚴 骑行

| 项目 | 说明 |
|------|------|
| 路线 | 梅灵路，杭州最美骑行路线之一 |
| 距离 | 全程约 6km |
| 难度 | 坡度平缓，适合休闲骑 |
| 租车 | 共享单车即可 |

### 🥾 徒步

- 可以连接 [[十里琅珰]] 徒步线路
- 从梅家坞上山 → 五云山 → 云栖竹径（经典环线，约 3h）

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 茶山 | 层层叠叠的茶园 | 广角，展示规模 |
| 茶室院子 | 茶杯+茶山+蓝天 | 俯拍茶杯，背景茶山 |
| 梅灵路 | 林荫骑行道 | 骑行中的抓拍 |
| 采茶 | 如果看到茶农在采茶 | 远距离拍，不要打扰 |
| 茶叶特写 | 嫩绿的龙井茶芽 | 近景微距 |

> 💡 **穿搭建议**：棉麻/白裙/浅色系——在绿色茶山背景中特别出片。

## 🟢 串联建议

```
方案 1（茶山骑行线）：
灵隐寺附近 →[骑行 15min]→ 梅家坞（喝茶 1h）→[骑行 20min]→ 龙井村
    →[步行 1.5h]→ 九溪十八涧

方案 2（轻松版）：
公交至梅家坞 → 农家茶室喝茶 2h → 沿梅灵路散步 → 返程

方案 3（徒步环线）：
梅家坞 → 上山 → 五云山 → 云栖竹径（约 3h，中等难度）
```

## 与龙井村对比

| 维度 | [[梅家坞]] | [[龙井村]] |
|------|-----------|-----------|
| 知名度 | 较低 | 极高 |
| 游客量 | 🟢 少 | 🟡 中等 |
| 茶室价格 | 更实惠 | 偏贵 |
| 茶叶品质 | 同级（都是西湖龙井）| 同级 |
| 环境 | 茶山+农家 | 村落+茶山 |
| 适合 | 想安静喝茶的人 | 想打卡的人 |

## ⚠️ 五一提醒

- 梅家坞五一人不多——大家都去龙井村了
- 🟢 如果龙井村人太多 → 改来梅家坞，体验更好
- 农家茶室五一可能比平时贵一点——先问价
- 骑行梅灵路五一人不多，但注意来往车辆
- 蚊虫较多——带驱蚊液
- 山区温度比市区低 2-3°C——带件薄外套
