# 平湖秋月

西湖十景之一。白堤中段的临水平台，以秋夜赏月闻名——湖面如镜，月光、灯光、塔影交相辉映。五一非最佳季节（没月亮也没秋天），但仍是白堤徒步的必经打卡点和绝佳拍照位。

## 基本信息

- **门票**：免费
- **位置**：白堤中段，孤山脚下
- **最佳季节**：中秋前后（9-10 月）月圆之夜
- **游览时间**：15-30min（打卡拍照）/ 1h（坐着发呆看湖）
- **交通**：白堤步行到达

## 核心看点

| 看点 | 说明 |
|------|------|
| 临水平台 | 伸入湖面的平台，三面环水 |
| 御碑亭 | 清康熙题"平湖秋月"碑 |
| 曲桥 | 连接孤山的小桥 |
| 湖景 | 正对西湖最开阔的湖面 |

> 💡 五一虽然看不了月亮，但**临水平台看西湖的视角是整条白堤最好的**——三面环水，视野极其开阔。

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 临水平台 | 三面环水，背景是城隍阁+雷峰塔 | 广角，把塔和水都收进去 |
| 廊亭飞檐 | 飞檐+湖景 | 框架构图 |
| 御碑亭 | "平湖秋月"碑 | 人物+碑合影 |
| 曲桥 | 桥+湖+孤山 | 侧拍，纵深感 |

> 💡 **最佳时间**：清晨 6:30-7:00 人最少+晨雾出片 / 傍晚 17:00-18:00 光线最好

## 🟢 串联建议

```
断桥残雪 →[步行 15min]→ 平湖秋月（拍照 15min）→[步行 5min]→ 孤山
    → 浙江省博物馆（孤山馆）→ 西泠印社
    → 过西泠桥 → 岳王庙 → 苏堤北口
```

> 这就是经典的**白堤徒步线路**——全程约 2.5km，步行 1.5h（含停留 2-3h）。

## 🍜 补给

- 白堤沿线有自动贩卖机和流动小摊
- 建议自带水（景区价格翻倍）
- [[孤山]]脚下有小卖部
- 岳庙附近有面馆

## ⚠️ 五一提醒

- 平湖秋月是白堤上人最密集的点之一——但只是路过打卡，不需停留太久
- 🟢 **不要挤在平台上拍照**——往两侧走走，人少且视角一样好
- 清晨来最值——6:30 的平湖秋月几乎没人，晨雾+湖面=仙境
- 如果白堤人太多 → 从平湖秋月拐入孤山——立刻人少一半
