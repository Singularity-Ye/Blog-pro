# 北高峰

杭州城西最高峰（海拔 314m）。山顶灵顺寺——"天下第一财神庙"，求财极灵。可俯瞰西湖全景和龙井茶田。是灵隐景区的"隐藏玩法"——大部分游客只去灵隐寺，不知道山上还有财神庙。

## 基本信息

| 项目 | 详情 |
|------|------|
| **山顶寺庙** | 灵顺寺（天下第一财神庙），门票 **¥8** |
| **索道上行** | **¥30/人**（约 5-10min 直达山顶） |
| **索道下行** | **¥20/人** |
| **往返套票** | **¥50/人** |
| **索道运营** | 7:30 开始，上行票 16:00 后停售 |
| **游览时间** | 2-3h（含上下山+山顶） |

## 🚶 三种上山方式

| 方式 | 耗时 | 适合 | 五一注意 |
|------|:--:|------|------|
| 🚡 **索道** | 5-10min | 懒人首选 | ⚠️ 旺季排队 1-3h |
| 🥾 **法华寺步道** | 20-30min | 老杭州私藏，路陡但快 | 🟢 人少，推荐！ |
| 🥾 **索道旁步道** | 40-50min | 经典徒步，经韬光寺 | 🟡 人适中 |

> 🟢 **推荐组合**：索道上+步行下（经韬光寺→永福寺→灵隐寺，全程下行 40-60min）

| 财神 | 职能 | 适合 |
|------|------|------|

## 🟢 串联建议

```
方案 1（经典组合）：
法华寺步道上山（20min）→ 灵顺寺（30min）
    → 索道下山（5min）→ 灵隐寺（2h）→ 法喜寺（1h）

方案 2（徒步全穿）：
索道上山（5min）→ 灵顺寺（30min）
    → 步行下山 → 韬光寺 → 永福寺 → 灵隐寺（全程 1.5h）

方案 3（轻松版）：
索道往返（¥50）→ 灵顺寺拜财神（30min）→ 下山
```

## ⚠️ 五一提醒

- 去北高峰**不走灵隐路**（堵死）→ 走**法华寺步道**上山（地铁 3 号线东岳站 D 口）
- 索道五一排队 1-3h——**7:30 开门就去**或直接步行上山
- 🟢 山顶俯瞰西湖全景，是避开山下人潮的好选择
- 下山后串联 [[灵隐寺]] 或 [[法喜寺]]
- 山顶风大+温度低——带件外套
- 穿运动鞋——步道有些陡
