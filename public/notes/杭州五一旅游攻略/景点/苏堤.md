# 苏堤

西湖十景之首"苏堤春晓"。北宋苏轼任杭州知州时疏浚西湖所筑。六座石桥贯穿南北，全长约 2.8km——是西湖最经典的徒步线路，也是五一最拥挤的路段。

## 基本信息

- **门票**：免费
- **长度**：约 2.8km（步行约 40-60 分钟，不停留的话）
- **六桥**：映波桥 → 锁澜桥 → 望山桥 → 压堤桥 → 东浦桥 → 跨虹桥
- **起点**：北端接 [[岳王庙]]/[[曲院风荷]]，南端接花港观鱼/[[南山路]]
- **游览时间**：1-1.5h（含拍照停留）

## 🔴 骑行禁令

苏堤和白堤**永久禁止自行车通行**（非五一特例）。入口有保安拦截。

| 堤道 | 自行车 | 步行 | 电瓶车 |
|------|:---:|:---:|:---:|
| 苏堤 | ❌ | ✅ | ✅ |
| 白堤 | ❌ | ✅ | ✅ |
| 杨公堤 | ✅ | ✅ | ✅ |

> 💡 想骑车游西湖 → 走**杨公堤**（绿荫更浓，人少景美）


## 🍜 堤上补给

- 苏堤本身没有商店——**提前买水和零食**
- 南口 → [[净慈寺素烧鹅]]（步行 5min，¥10/份）
- 北口 → [[岳王庙]]周边小吃
- 花港观鱼旁有茶馆

## ⚠️ 五一提醒

- 🔴 苏堤是五一西湖最拥挤的路段——10:00-15:00 堤上人贴人
- 🟢 最佳策略：**7:00 前走完** 或 **16:30 后去**（看雷峰夕照）
- 电瓶车（¥40/人单程）可以在苏堤上坐——如果走不动了
- 堤上没有遮阳——五月杭州太阳已经很毒，带帽子+涂防晒
- 六座桥都有上下坡——老人/小孩注意节奏
- 💡 如果人太多走不动 → 从中间任何一座桥拐出苏堤，沿杨公堤走——人少 80%
