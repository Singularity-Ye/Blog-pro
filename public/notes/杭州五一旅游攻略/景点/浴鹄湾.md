# 浴鹄湾

杨公堤旁的隐藏秘境。与 [[苏堤]] 仅一墙之隔，却是两个完全不同的世界——苏堤挤成"人从众"，浴鹄湾里只有茅草屋、石板桥和野鸭白鹭。**零旅行团。**

## 基本信息

- **门票**：免费
- **位置**：西湖区杨公堤（靠近 [[花港观鱼]] 和 [[太子湾公园]]）
- **交通**：从苏堤南口沿杨公堤步行约 15min；公交 197/318/1314 路「浴鹄湾」站
- **游览时间**：30-60min
- **最佳时间**：清晨（6:30-8:00，晨雾+无人）或傍晚（16:00-17:30，金色光线）

## 核心看点

| 看点 | 说明 |
|------|------|
| 霁虹桥 | 茅草屋顶长廊+水面倒影，婚纱摄影师最爱 |
| 石板曲桥 | 贴着水面的石板桥 |
| 野鸭白鹭 | 水鸟出没，生态环境极好 |
| 竹林幽径 | 幽静小路 |
| 茅草屋 | 湖畔茅草亭，水墨画既视感 |

## 📸 拍照机位

| 机位 | 拍什么 | 最佳时间 | 技巧 |
|------|--------|:--:|------|
| **霁虹桥长廊对面** | 廊桥+茅草屋顶+水面倒影 | 清晨/傍晚 | 对称构图，水面倒影 |
| **石头台阶旁** | 小瀑布+人像 | 全天 | 低角度，拍水流 |
| **武状元坊亲水平台** | 用桥洞做框景拍湖面 | 下午 | 框架构图 |
| **水边长椅** | 电影感构图 | 傍晚逆光 | 人物剪影+湖面 |
| **竹林小径** | 竹子+光斑 | 上午 | 侧光，光斑感 |

> 霁虹桥是婚纱摄影师最爱的取景地之一。

> 💡 **穿搭建议**：白色/浅色/素色衣服——在绿色+碧水背景中特别出片。避免花哨图案。

## 周边

- [[乌龟潭]]：步行 10min（更安静，10:00 后放水）
- [[花港观鱼]]：步行 15min
- [[郭庄]]：步行 10min（¥10，江南园林喝茶看湖）
- [[太子湾公园]]：步行 15min

## 注意事项

- 🦟 水边蚊虫多，带驱蚊液
- 🚻 附近无洗手间（花港观鱼或郭庄有）
- 🍜 无餐饮摊贩，自带水
- 📸 穿浅色/素色衣服更出片
- ⚠️ 水边没有护栏，注意安全——尤其带小朋友
