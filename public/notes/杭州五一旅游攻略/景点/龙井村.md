# 龙井村

西湖龙井茶的核心产区。"茶乡第一村"，四周群山环抱、茶园层叠。是 [[九溪十八涧]] 和 [[十里琅珰]] 徒步的枢纽节点。五一虽已过采茶季，但茶园依然壮观。

## 基本信息

- **门票**：免费
- **交通**：公交 27 路「龙井村」站
- **游览时间**：1-2h（喝茶+逛茶园）/ 半天（含徒步）
- **最佳季节**：3 月下旬-4 月（明前茶季），五一已过但茶山依然绿

## 🍵 品茶避坑

| 做法 | 🔴/🟢 |
|------|--------|
| 路边大妈拉你去"免费品茶" | 🔴 99% 购物陷阱，喝完必推销 |
| 门口有"茶"字招牌的茶农家 | 🟢 明码标价，可泡一杯喝 |
| 村内正规茶馆 | 🟢 环境好，价格透明 |
| 路边散装茶叶 | 🔴 真假难辨 |

> **茶叶参考价**：普通 ¥300/斤，好的 ¥500/斤。几十块一斤的"龙井"必然是假的。

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 龙井村观景亭 | 茶田+远山全景 | 广角，层叠茶田 |
| 茶园小径 | 走在茶田间 | 侧拍，人物+茶田 |
| 茶杯+茶山 | 农家院子里喝茶 | 俯拍茶杯，背景茶山 |
| 石板路 | 村中古道 | 纵深感构图 |
| 十里琅珰入口 | 上山步道+茶田 | 展示徒步感 |

> 💡 **穿搭建议**：白色/浅色棉麻——在绿色茶山背景中特别出片。避免绿色（会融入背景）。

## 🥾 徒步枢纽

龙井村是杭州经典徒步路线的中转站：

```
方案 1（经典下坡线）：
法喜寺 →[打车 15min 或步行 40min]→ 龙井村
    → 十里琅珰（茶山徒步，2-3h）→ 九溪十八涧（1.5-2h）→ 九溪公交站

方案 2（轻松版）：
龙井村（喝茶 1h）→ 步行 1.5h → 九溪十八涧（下坡，轻松）

方案 3（上坡版）：
九溪公交站 → 步行 2-2.5h → 九溪十八涧 → 龙井村（一路上坡）
```

> 🟢 推荐从龙井村下坡走九溪，节省体力。

## ⚠️ 五一提醒

- 龙井村五一人比平时多，但比西湖好很多
- 🟢 **不要接受路边大妈的"免费品茶"邀请**——99% 是购物陷阱
- 如果想买正宗西湖龙井 → 找有门牌的茶农或正规茶馆
- 村内没有便利店——提前买水
- ❌ 禁止踩踏茶田（茶农的命根子）
- 🥾 穿防滑鞋，石板路青苔多
- 🦟 带驱蚊液，山里蚊子多
