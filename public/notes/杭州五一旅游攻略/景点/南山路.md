# 南山路

西湖东南岸的景观大道，杭州最美的梧桐大道之一。两侧法国梧桐遮天蔽日，沿途串联 [[净慈寺]]、[[雷峰塔]]、[[太子湾公园]]、[[苏堤]] 南口、中国美院南山校区——是西湖环线中**步行最舒适、文化密度最高**的一段。

## 基本信息

- **长度**：约 3.5km（湖滨→苏堤南口）
- **门票**：免费（沿途景点单独收费）
- **交通**：地铁 1 号线至定安路站步行 10min / 龙翔桥站步行 5min（⚠️ 限流）
- **步行时间**：纯走 40min，边走边逛 1.5-2h

## 南山路集群（从北到南）

```

    ↓ 步行 5min

```

## 🟢 为什么南山路是五一最佳步行路线

| 对比 | 白堤/苏堤 | 南山路 |
|------|----------|--------|
| 树荫 | ❌ 无遮挡 | ✅ 梧桐全覆盖 |
| 拥挤度 | 🔴 挤爆 | 🟡 中等 |
| 咖啡/茶馆 | ❌ 几乎没有 | ✅ 沿路很多 |
| 文化密度 | 中 | 高（寺庙+塔+美术馆+公园） |
| 随时休息 | ❌ 只能坐石椅 | ✅ 随时找店坐下 |

## ⚠️ 五一提醒

- 龙翔桥地铁站限流 → 从 **定安路站** 或 **凤起路站** 步行前往
- 南山路本身不堵（步行为主），但延安路/湖滨一带人多
- 傍晚时分西湖边人最多——想安静就上午去
- 共享单车在南山路可以骑（不在禁骑区）
