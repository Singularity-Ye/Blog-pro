# 河坊街

明清仿古商业街，杭州最知名的旅游步行街之一。各种杭帮特产、小吃、手工艺品汇集。但说实话——**主街商业化严重，真正的杭州味道在旁边的大马弄和中山南路。**

## 基本信息

- **门票**：免费
- **位置**：**上城区**（⚠️ 原图错误归入西湖区）
- **开放时间**：全天（店铺约 10:00 - 22:00）
- **游览时间**：30-60min（主街快速过）/ 2-3h（含大马弄+中山南路）
- **交通**：地铁 7 号线吴山广场站步行 3min

## 🏮 老城区集群

河坊街与 [[南宋御街]] 十字相交，是杭州老城区核心：

```
河坊街（东西向）× 南宋御街（南北向）= 杭州老城核心区
    ↓
吴山广场 → 吴山（城隍阁 ¥30）
    ↓
中山南路 → 大马弄（老底子美食天堂）
    ↓
鼓楼 → 中山南路 → 游埠豆浆 / 孙奶奶葱包烩 / 菊英面馆
```

## ⚠️ 五一提醒

- 🔴 河坊街主街五一人山人海——但旁边的**大马弄人少很多**
- 🟢 策略：主街快速过（20min）→ 拐进大马弄和中山南路（真正的美食天堂）
- 主街小吃价格偏贵，拐进巷子吃正宗且便宜一半
- 茶叶、丝绸等特产建议去**世纪联华**或正规店买，路边摊假货多
- 如果太挤 → 转入南宋御街，人少一半
- 附近 [[杭州博物馆]] 需要提前预约——如果约到了可以一起逛
