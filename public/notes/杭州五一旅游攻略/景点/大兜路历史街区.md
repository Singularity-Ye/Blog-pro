# 大兜路历史街区

京杭大运河畔的历史文化街区，比 [[小河直街]] 更安静、更有生活气息。保留明清运河漕运建筑风貌。是运河沿线"最本地"的街区——没有网红咖啡馆，只有老居民和小餐馆。

## 基本信息

- **门票**：免费
- **位置**：拱墅区，运河西岸
- **游览时间**：30-60min
- **交通**：[[拱宸桥]] 步行 15min / [[香积寺]] 步行 10min / 地铁 3 号线香积寺站步行 5min

## 核心看点

| 看点 | 说明 |
|------|------|
| 明清漕运建筑 | 白墙黛瓦，沿河而建 |
| 运河河畔 | 可以在河边散步/发呆 |
| 老居民区 | 最真实的杭州老城生活 |
| 香积寺铜殿 | 街区北端，步行 5min |
| 胜利河美食街 | 街区附近，龙虾啤酒节 |

## 🎭 五一活动

**「兜来放风计划」**（5/1-5）：文创市集+轻食+手作工坊。

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 运河河畔 | 运河+对岸民居+柳树 | 广角，对称构图 |
| 老建筑门面 | 清末民初建筑 | 竖构图，建筑细节 |
| 石板路 | 老街+灯笼 | 纵深感 |
| 傍晚 | 灯笼亮起+河面倒影 | 蓝调时刻 |

> 💡 这里是拍"老杭州生活"最好的地方——没有商业化的痕迹。


## ⚠️ 五一提醒

- 大兜路是运河沿线人最少的街区——**如果你只想安静散步，来这里**
- 「兜来放风计划」市集五一人会多一些，但整体还是很舒适
- 没有太多商业设施——提前买水
- 适合安排在行程末尾：逛完小河直街/拱宸桥→大兜路散步→胜利河晚餐
- 傍晚灯光效果好——如果行程允许，待到 18:00-19:00
