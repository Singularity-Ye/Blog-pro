# 花港观鱼

西湖十景之一。位于 [[苏堤]] 南端，以红鲤鱼池、牡丹园和临湖长廊闻名。是苏堤徒步的终点/中转站——也是通往 [[浴鹄湾]]/[[郭庄]] 秘境线的入口。

## 基本信息

- **门票**：免费
- **位置**：苏堤南端，[[南山路]] 与杨公堤之间
- **游览时间**：20-40min
- **交通**：苏堤步行到达 / 公交至花港观鱼站
- **最佳季节**：春季牡丹花期（五一花期已过，但鱼池和园林全年可看）

## 核心看点

| 看点 | 说明 | 推荐 |
|------|------|:--:|
| 🐟 **红鲤鱼池** | 上千条锦鲤，投喂已禁止（鱼太胖了）| ⭐⭐⭐⭐⭐ |
| 🌸 **牡丹园** | 五一花期已过 | ⭐⭐ |
| 🏛️ **林徽因纪念碑** | 花港观鱼内有林徽因剪影镂空雕塑 | ⭐⭐⭐⭐ |
| 🏚️ **临湖长廊** | 看苏堤和雷峰塔的绝佳位置 | ⭐⭐⭐⭐⭐ |
| 🌳 **大草坪** | 可以躺平休息 | ⭐⭐⭐ |

> 💡 **林徽因纪念碑**是花港观鱼的隐藏打卡点——剪影镂空雕塑，阳光下投影出林徽因的轮廓，非常有意境。

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 红鲤鱼池 | 千条锦鲤+水 | 俯拍鱼群，等鱼聚拢 |
| 林徽因纪念碑 | 剪影雕塑+阳光 | 逆光拍投影 |
| 临湖长廊 | 长廊+湖+雷峰塔 | 框架构图，长廊做前景 |
| 大草坪 | 草坪+大树+远山 | 躺拍，天空+树冠 |
| 苏堤方向回望 | 看苏堤延伸 | 经典构图 |

## 🟢 串联建议

花港观鱼是苏堤→南山路的关键中转：

```
方案 1（苏堤→秘境切换）：
苏堤（2.8km）→ 花港观鱼（休息 20min）→ 沿杨公堤步行 15min
    → 浴鹄湾 → 乌龟潭 → 郭庄

方案 2（经典收尾）：
苏堤 → 花港观鱼 → 步行 5min → 净慈寺（买素烧鹅）
    → 步行 5min → 雷峰塔

方案 3（轻松版）：
花港观鱼（看鱼+草坪休息）→ 沿南山路散步
    → 步行 15min → [[西湖游船]]（花港码头上船）
```

## 🍜 周边补给

- 花港观鱼门口有自动贩卖机和流动摊
- 苏堤南口附近有小餐馆（价格偏贵，建议自带）
- [[净慈寺素烧鹅]]：步行 10min（南山路方向，¥7/份）
- 花港观鱼内有大草坪——自带零食铺野餐垫也很棒

## ⚠️ 五一提醒

- 免费景点，上午 10 点后人多
- 🟢 不要在这里停留太久——把时间留给浴鹄湾/郭庄
- 不要喂鱼（鱼已经很胖了，喂食会撑死）
- 可直接从花港观鱼转入杨公堤秘境线（浴鹄湾/乌龟潭/郭庄）
- 如果想坐船 → 花港码头是 6 个码头之一（[[西湖游船]]）
- 💡 最佳策略：苏堤走完→花港观鱼休息 20min→拐入浴鹄湾秘境线
