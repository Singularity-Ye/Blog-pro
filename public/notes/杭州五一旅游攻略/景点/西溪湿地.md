# 西溪湿地

国家 5A 级旅游景区，中国首个国家湿地公园。电影《非诚勿扰》取景地。距西湖仅 5km，却是完全不同的生态世界——水道纵横、芦苇丛生、白鹭成群。

## 基本信息

- **门票**：¥80（五一可能有折扣，关注官方公告）
- 🆓 **免费区域**：绿堤、福堤、河渚街、高庄无需门票
- 🆕 **春假特惠**（至 5/6）：中小学生「门票+电瓶船」套票 **¥67.8**（原价 ¥140）
- **电瓶船**：¥60/人（强烈建议买！园区太大，纯走路会崩溃）
- **电瓶车**：¥20/人（不想走路的备选）
- 💡 **套票建议**：门票+游船+电瓶车 约 ¥160，比单买划算
- **摇橹船**：¥100/h（限坐 6 人，更有意境）
- **推荐游览时间**：半天（4-5h）
- **交通**：地铁 3 号线「西溪湿地南」站 / 地铁 19 号线「西溪湿地北」站
- **开放时间**：8:00 - 17:30（旺季可能延长至 18:00）

## 🔴 必买车/船票！

西溪湿地面积约 11.5km²。**纯靠腿走会迷失在芦苇荡里走到崩溃**。建议买「门票+电瓶船」套票。

## 分区

| 区域 | 特点 | 适合 | 游览时间 |
|------|------|------|:--:|
| **东区**（一期）| 核心景区，电影取景地，摇橹船 | 首次游览 | 3-4h |
| **西区**（洪园）| 更野生，人少，洪氏宗祠 | 深度游/避人流 | 2-3h |
| **福堤** | 免费区域，连接东西区 | 骑行/散步 | 1-2h |

## 📸 拍照攻略

| 机位 | 说明 | 技巧 |
|------|------|------|
| 烟水渔庄 | 水面+芦苇+渔船 | 广角，水墨画感 |
| 秋雪庵 | 湖中孤岛+芦苇 | 远拍，意境感 |
| 摇橹船上 | 船夫+水道+两岸 | 低角度，船上视角 |
| 芦苇丛 | 芦苇+水鸟 | 逆光拍芦苇穗 |
| 河渚街 | 小桥流水+白墙 | 侧拍，江南水乡 |

> 💡 **穿搭建议**：棉麻/白裙/汉服——湿地背景太适合古风了。

## 🟢 推荐路线

```
方案 1（经典东区半日）：
南入口 → 电瓶船至深潭口（20min）→ 河渚街（30min）
    → 步行至烟水渔庄（20min）→ 摇橹船（1h）
    → 秋雪庵（20min）→ 电瓶船返回（20min）

方案 2（免费区域）：
福堤入口 → 绿堤散步（1h）→ 河渚街（小吃 30min）
    → 高庄（30min）→ 返回（全程免费）

方案 3（亲子线）：
东区电瓶船（看水鸟）→ 河渚街（吃小吃）
    → 福堤骑行 → 高庄看花
```

## 🟢 五一价值

- 人比西湖**少很多**（名气不如西湖，但体验不输）
- 水道+芦苇+白鹭 = 完全不同的杭州
- 五一期间有《今夕共西溪》实景演出（夜间）
- 🆓 免费区域（绿堤/福堤/河渚街/高庄）不买票也能玩 2-3h

## ⚠️ 五一提醒

- 西溪湿地五一人比西湖少——但热门区域（摇橹船/河渚街）还是排队
- 🟢 **8:00 开园就去**——避开 10:00+ 的人流
- 摇橹船五一可能排队 30-60min——早点去或改电瓶船
- 🦟 湿地蚊子极多，带**强效驱蚊液**
- ☀️ 遮阳帽/伞必备（部分路段无树荫）
- 🥾 穿舒适的鞋（全程步行量大）
- 📸 烟水渔庄、秋雪庵是最出片点
- 🆓 不想花钱 → 走免费区域（绿堤+福堤+河渚街+高庄）也能玩 2-3h
