## 🚇 Step by Step 详细行程

### 第一程：下沙 → 嘉里中心

```
9:00  生活三区出门
        ↓ 🚶步行 30min（或骑车 10min）
9:30  文泽路站 A口进
        ↓ 🚇1号线（往湘湖方向）
        经：高沙路→金沙湖→客运中心→火车东站→武林广场
        ↓ ⚠️ 在凤起路站下车！不要到龙翔桥！
10:10 凤起路站 D出口
        ↓ 🚶步行 5min（沿延安路往南）
10:15 嘉里中心 到达
```

> 📍 **凤起路站 D出口** 出来就是延安路，往南走 5 分钟就到嘉里中心。嘉里中心在延安路和凤起路交叉口东南角。

---

### 第二程：嘉里中心 · MR.ROSE 粉红先生（10:15-11:00）

```
10:15 🎨 MR.ROSE 粉红先生 杭州首展
        └── 📍 嘉里中心 1F 中庭/户外广场
        └── 💰 免费
        └── 法国艺术家 Philippe Katerine 的巨型粉色雕塑
        └── 📸 机位：
              ① 雕塑正面全身照（手机放低仰拍，显高）
              ② 侧面互动照（可以"拥抱"雕塑）
              ③ 嘉里中心二楼俯拍（全景构图）
        └── 🕐 逛 30-45min，拍照打卡
        └── 🛍️ 嘉里中心本身也是不错的商场，可以顺便逛逛
```

> 💡 **拍照技巧**：粉色雕塑在阳光下颜色最正，今天多云可能偏灰——如果有阳光赶紧拍。穿白色/浅色衣服和粉色最搭。

---

### 第三程：杭州大厦 · 毛毛怪展（11:00-11:30）

```
11:00 🚶 → 杭州大厦
        └── 从嘉里中心步行约 10min
        └── 沿延安路往南 → 武林广场方向
        └── 杭州大厦在武林广场东北角

11:10 🎨 杭州大厦「毛毛怪来袭」毛绒主题展
        └── 📍 杭州大厦商场内（具体楼层关注现场指引）
        └── 💰 免费
        └── 巨型毛绒怪兽装置 + 互动拍照点
        └── 📸 机位：和毛绒怪兽"互动"合影
        └── 🕐 逛 20-30min
        └── 🛍️ 杭州大厦是杭州最高端的商场，逛逛也无妨
```

### 第四程：新白鹿 午餐（11:30-13:00）⭐ 今天重头戏

```
11:20 📱 大众点评 APP → 搜索「新白鹿 武林广场店」→ 线上取号
        └── ⚡ 如果显示前方 >20 桌 → 先去杭州大厦逛着等叫号
        └── ⚡ 如果显示 <10 桌 → 赶紧走过去

11:30 🍜 新白鹿（武林广场店）
        └── 📍 武林广场附近（杭州大厦步行 5min）
        └── 💰 人均 ¥50-70
        └── 📝 必点：
              ✅ 糖醋里脊 ¥28（招牌！）
              ✅ 蛋黄鸡翅 ¥32
              ✅ 白鹿招牌炒饭 ¥22
              ✅ 西湖牛肉羹 ¥28
              ───────
              💰 2人合计约 ¥110（人均 ¥55）
        └── 🕐 吃 1-1.5h，不赶

13:00 吃完休息
```

> ⚠️ **取号关键时间点**：
> - 如果 **10:30 到店** → 基本不用排队，第一批进去
> - 如果 **11:00-11:30 到店** → 线上取号，等 20-40 分钟
> - 如果 **12:00 后到** → 等 1-2 小时，建议改去西湖银泰店（下午顺路）

### 第五程：湖滨商圈逛展（13:00-15:30）

```
13:00 🚶 从武林广场 → 湖滨银泰 in77
        └── 步行约 15min（沿延安路往南直走）
        └── 或 🚇 凤起路→龙翔桥 1站（⚠️ 但龙翔桥可能限流）
        └── 🟢 建议步行！延安路本身就是商业街，边走边逛

13:15 🎨 in77 夏日么么集
        └── 📍 湖滨银泰 in77（东坡路）
        └── 💰 免费
        └── 夏日泳池主题装置 + 创意市集 + 现场 DJ
        └── 📸 机位：泳池主题装置（夏日风，适合拍照）
        └── 🕐 逛 30-45min

14:00 🚶 → 西湖银泰
        └── 从 in77 步行约 5min（延安路上继续往南）

14:05 🎨 西湖银泰小黄人主题展
        └── 📍 西湖银泰 1F
        └── 💰 免费
        └── 巨型小黄人装置 + 互动拍照区 + 限定扭蛋
        └── 📸 机位：和巨型小黄人合影（蹲下来拍更萌）
        └── 🛍️ 可以买个扭蛋 ¥20-30 作纪念
        └── 🕐 逛 20-30min

14:30 ☕ 休息时间
        └── 西湖银泰 / in77 内找家咖啡馆坐下
        └── 或走到西湖边（步行 5min）找个长椅休息
        └── 💰 ¥25-35
```

> 💡 **in77 和西湖银泰** 其实就隔一条延安路，步行 3 分钟。两个商场可以串在一起逛。

### 第六程：西湖边 + 太子湾（15:00-16:30）可选

```
15:00 🚶 → 西湖边散步
        └── 从银泰往西走 5min 就到西湖
        └── 沿南山路往南走 → 长桥公园 → 太子湾方向

        └── 💰 免费

        └── 🕐 逛 30-45min
        └── ⚠️ 如果腿酸可以跳过，直接去河坊街

16:00 🚶 → 河坊街方向
        └── 从太子湾步行约 20min
        └── 或骑行 10min
```

> 🟢 **灵活调整**：如果逛商场已经很开心了，15:00 这段可以跳过。直接从银泰走去河坊街（步行 15min）。

## 🚨 应变决策树

### 新白鹿排队太久怎么办？

```
Q: 线上取号显示前方 30+ 桌？
  → 方案 1：等！先去杭州大厦/嘉里中心逛着，APP 看排队进度
  → 方案 2：换去西湖银泰店（下午逛银泰时吃，避开午高峰）
  → 方案 3：改吃「外婆家」（同类型杭帮菜，人均 ¥60-80）
  → 方案 4：改吃「弄堂里」（杭帮菜，环境好，人均 ¥60-80）

Q: 中午错过了新白鹿？
  → 下午逛完银泰后去「新白鹿 西湖银泰店」吃晚餐！
  → 17:00 前到，晚餐第一波
```

### 嘉里中心 MR.ROSE 排队/找不到？

```
Q: 粉红先生雕塑前排队拍照？
  → 先去旁边的杭州大厦毛毛怪展（步行 10min），回来再拍
  → 或者换个角度拍——侧面/背面也很出片

Q: 找不到 MR.ROSE？
  → 嘉里中心 1F 中庭或户外广场
  → 问商场服务台
  → 看到粉色就对了
```

### 下雨了怎么办？

| 雨量 | 方案 |
|------|------|
| 🌧️ 小雨 | 照走。商场之间有骑楼/遮雨棚，淋不到多少。带折叠伞即可。 |
| 🌧️ 中雨 | 砍掉太子湾+西湖边（第六程）。从银泰直接走去河坊街（有骑楼）。 |
| 🌧️ 大雨 | 全天室内：嘉里中心→杭州大厦→新白鹿→in77→西湖银泰→地铁回家。完美避雨。 |
| 🌧️ 傍晚雨停 | 去西湖边！雨后空气最好，人也少。 |

> 🟢 **路线 D 最大的优势就是不怕天气**——几乎全程室内/商场，下雨天的最佳选择。

### 逛展太无聊了？

```
Q: 商场展看腻了？
  → 选项 1：去 [[杭州海底世界]]（南山路，¥120-150，室内水族馆）
  → 选项 2：去 [[夜游西湖]]（西湖边散步，免费）
  → 选项 3：去 [[武林夜市]]（吃吃喝喝）
  → 选项 4：地铁回下沙，金沙天街逛逛（完全不累）
```

### 太累了走不动？

```
Q: 腿酸得不行？
  → 砍掉第六程（太子湾+西湖边）和第八程（夜游）
  → 在河坊街吃完晚饭直接回家
  → 17:30 从定安路出发 → 18:30 到文泽路 → 19:00 到家
```

### 龙翔桥限流了？

```
Q: 想去 in77 但龙翔桥地铁限流？
  → 从凤起路站 D出口 往南步行 10min 到嘉里中心/武林广场
  → 从定安路站 往北步行 10min 到西湖银泰/河坊街
  → 绝对不要在龙翔桥站上下车！
```
