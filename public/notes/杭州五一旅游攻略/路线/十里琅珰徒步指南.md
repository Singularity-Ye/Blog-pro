# 十里琅珰徒步指南

杭州徒步的"天花板"级路线。串联溪涧、古村、茶山、竹海四种地貌。从 [[龙井村]] 经十里琅珰到 [[九溪十八涧]]。

## 🗺️ 推荐路线

### 经典线（新手首选）

```
九溪公交站 → 九溪烟树 → 九溪十八涧 → 龙井村 → 十里琅珰 → 梅家坞
```

| 段 | 里程 | 耗时 | 特点 |
|----|------|------|------|
| 九溪烟树 → 九溪十八涧 → 龙井村 | 3km | 50min-1.5h | 溪水相伴、踩石过溪 |
| 龙井村 → 十里琅珰山顶 | 2km | 1.5h | 千级台阶、茶园全景 |
| 山顶 → 梅家坞（下山） | 3km | 1h | 平缓下山 |

**全程：约 8-10km / 3-5 小时**

### 进阶线（体力充足）

```
九溪公交站 → 九溪烟树 → 龙井村 → 十里琅珰 → 五云山 → 真际寺 → 云栖竹径
```

全程约 11-12km，4-5 小时，最高海拔 370m。

## ⚠️ 装备必带

| 物品 | 原因 |
|------|------|
| 🥾 防滑徒步鞋 | 石板路青苔多，雨后极滑。**禁穿拖鞋/高跟鞋** |
| 🦯 登山杖 | 十里琅珰千级台阶，竹杖 ¥3 能救命 |
| 🧴 强效驱蚊液 | 山里蚊子极多 |
| ☀️ 高倍防晒 | 茶山无遮挡 |
| 💧 2 瓶水 | 十里琅珰山顶**无商店** |
| 🔋 充电宝 | 全程拍照多 |

## ⏰ 时间铁律

- 🕘 **9:00 前出发**，确保天黑前下山
- 🕑 **最晚 14:00 前上山**
- 🌙 **绝对不走夜路！**茶山段无路灯

## 🚫 禁止

- ❌ 踩踏茶田（茶农命根）
- ❌ 乱扔垃圾
- ❌ 无人机（茶园禁飞区）

## 📸 拍照点

1. 九溪烟树瀑布（"小九寨沟"）
2. 九溪十八涧踩石过溪
3. 十里琅珰山顶千亩茶园全景
4. 龙井村观景亭（茶田+远山）

## 🔗 串联

```
法喜寺 → 龙井村 → 十里琅珰 → 九溪十八涧 → 九溪公交站
```

> 建议从龙井村往九溪方向走（一路下坡省体力）。
