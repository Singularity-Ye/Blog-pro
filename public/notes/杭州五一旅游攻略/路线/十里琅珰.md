# 十里琅珰

杭州最经典的茶山徒步路线。从 [[龙井村]] 到梅家坞/云栖竹径，千亩茶园绵延山脊。详见 [[十里琅珰徒步指南]] 获取完整装备清单和路线方案。

## Route Points

1. [[法喜寺]]
2. [[龙井村]]
3. 十里琅珰（茶山山脊）
4. [[九溪十八涧]]

## 快速信息

- **全程**：8-12km / 3-5 小时
- **起点**：[[龙井村]]
- **终点**：梅家坞 或 [[九溪十八涧]]
- **难度**：中等（千级台阶）

## 关键提醒

- 🥾 必须穿防滑徒步鞋
- 💧 山顶无商店，自带 2 瓶水
- 🕘 最晚 14:00 前上山
- 🌙 绝对不走夜路

## 串联

```
法喜寺 → 龙井村 → 十里琅珰 → 九溪十八涧 → 九溪公交站
```
