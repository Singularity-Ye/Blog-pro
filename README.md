# 个人博客网站

## 项目说明
这是一个基于 React 开发的个人博客网站，使用了现代化的动画效果和响应式设计。

## 技术栈
- React 18
- React Router v6
- Framer Motion (动画效果)
- Styled Components (样式管理)

## 项目结构 
src/
├── components/ # 可复用组件
│ ├── Layout/ # 布局组件
│ ├── Animations/ # 动画组件
│ └── Blog/ # 博客相关组件
├── pages/ # 页面组件
├── styles/ # 全局样式
├── utils/ # 工具函数
└── assets/ # 静态资源

## 主要功能
1. 博客文章列表展示
2. 文章详情页
3. 关于我页面
4. 项目展示页面
5. 联系方式页面

## 动画效果
- 页面切换过渡动画
- 文章列表滚动动画
- 图片加载动画
- 交互反馈动画
